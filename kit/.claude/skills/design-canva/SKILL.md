# Canva Connect

Integra com a conta Canva do usuario para criar e exportar designs.

## Quando Usar

Use quando o usuario preferir Canva em vez de HTML/CSS para criar visuais.

## Capacidades

- Criar designs novos (apresentacoes, posts sociais, logos)
- Preencher templates com conteudo (autofill)
- Buscar designs existentes na conta do usuario
- Exportar como PDF, PNG, JPG

## Workflow

1. Use as ferramentas MCP do Canva para interagir
2. Prefira templates existentes — mais rapido e consistente com a marca
3. Ao preencher, combine nomes de campos com placeholders do template
4. Exporte no formato mais util: PNG para social, PDF para documentos

## Requisitos

- Conta Canva (gratis ou paga)
- Autorizacao OAuth na primeira vez (popup no navegador)
- Autofill de templates requer plano pago do Canva

## Boas Praticas

- Busque templates antes de criar do zero
- Use assets da marca do usuario quando disponiveis
- Exporte em alta resolucao para producao
