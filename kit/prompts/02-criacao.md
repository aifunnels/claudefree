### 1. Gerador de Calendario Editorial Semanal

**Quando usar:** Toda segunda-feira (ou no inicio do mes) para planejar o conteudo da semana do cliente.

**Prompt:**
```
Voce e um social media estrategista. Preciso de um calendario editorial semanal completo.

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Nicho: [NICHO]
- Redes: [INSTAGRAM / TIKTOK / LINKEDIN / TODAS]
- Frequencia: [NUMERO] posts por semana
- Tom de voz: [FORMAL / INFORMAL / TECNICO / DESCONTRAIDO]
- Objetivo do mes: [VENDAS / AUTORIDADE / ENGAJAMENTO / LANCAMENTO]
- Produto/servico em destaque: [PRODUTO OU SERVICO]
- Data especial na semana (se houver): [DATA COMEMORATIVA, EVENTO, PROMO]

Me entregue um CALENDARIO SEMANAL com:

Para cada dia de postagem:
1. DIA E HORARIO sugerido
2. FORMATO (feed, reels, stories, carrossel, live)
3. PILAR DE CONTEUDO (educativo, entretenimento, venda, bastidores, prova social)
4. TITULO DO POST
5. COPY COMPLETA (com CTA)
6. DESCRICAO DA ARTE/VIDEO (briefing visual para o designer)
7. HASHTAGS (10-15 relevantes)

Ao final, inclua:
- STORIES DIARIOS: 3 sugestoes de stories para cada dia (enquete, caixinha, bastidores)
- REELS DA SEMANA: 2 ideias de reels com roteiro resumido (gancho, desenvolvimento, CTA)

Formate como tabela organizada por dia da semana.
```

**Exemplo de resultado esperado:** Tabela com 5-7 posts planejados, cada um com copy pronta, briefing visual, hashtags e stories diarios complementares.

### 2. Criador de Copy para Anuncio (5 Variacoes)

**Quando usar:** Ao montar campanhas de trafego pago e precisar de variacoes para testar.

**Prompt:**
```
Preciso de 5 variacoes de copy para anuncio pago. Cada variacao deve usar uma abordagem diferente.

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Produto/servico anunciado: [PRODUTO OU SERVICO]
- Preco: R$ [VALOR] (ou "orcamento sob consulta")
- Publico-alvo: [DESCRICAO DO PUBLICO]
- Plataforma: [META ADS / GOOGLE ADS / TIKTOK ADS]
- Formato: [IMAGEM / VIDEO / CARROSSEL]
- Objetivo da campanha: [LEADS / VENDAS / TRAFEGO / CADASTRO]
- Destino (CTA): [WHATSAPP / LANDING PAGE / FORMULARIO / SITE]
- Diferencial principal: [O QUE FAZ ESSE PRODUTO SER UNICO]

Crie 5 variacoes usando abordagens diferentes:

VARIACAO 1 — DOR
- Headline focada na dor/problema do publico
- Copy curta (max 3 linhas) + CTA direto

VARIACAO 2 — DESEJO
- Headline focada no resultado/transformacao
- Copy curta (max 3 linhas) + CTA direto

VARIACAO 3 — PROVA SOCIAL
- Headline com dado, numero ou depoimento
- Copy curta (max 3 linhas) + CTA direto

VARIACAO 4 — URGENCIA
- Headline com escassez ou prazo
- Copy curta (max 3 linhas) + CTA direto

VARIACAO 5 — CURIOSIDADE
- Headline que gera curiosidade (sem clickbait)
- Copy curta (max 3 linhas) + CTA direto

Para cada variacao, entregue:
- Headline principal
- Texto do anuncio (primario)
- Descricao (se aplicavel)
- CTA do botao
- Sugestao de gancho visual para o criativo
```

**Exemplo de resultado esperado:** 5 copies completas com headlines, textos e CTAs variados, prontas para subir na plataforma de anuncios.

### 3. Gerador de Roteiro de Reels/Stories

**Quando usar:** Quando o cliente precisa gravar conteudo em video e voce precisa entregar o roteiro pronto.

**Prompt:**
```
Preciso de um roteiro completo para um video curto (Reels/TikTok/Stories).

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Quem vai gravar: [NOME DA PESSOA — dono, colaborador, influenciador]
- Tema do video: [ASSUNTO]
- Objetivo: [EDUCAR / VENDER / ENGAJAR / VIRALIZAR]
- Duracao alvo: [15s / 30s / 60s / 90s]
- Tom: [SERIO / DIVERTIDO / MOTIVACIONAL / TECNICO]
- CTA final: [O QUE QUER QUE A PESSOA FACA — seguir, comentar, clicar no link, salvar]

Me entregue:

1. ROTEIRO COMPLETO — formato teleprompter (frase por frase, facil de ler enquanto grava)

Estrutura:
- GANCHO (primeiros 3 segundos — frase que prende atencao)
- DESENVOLVIMENTO (conteudo principal — 3-5 pontos)
- CTA (chamada para acao final)

2. INSTRUCOES DE GRAVACAO
   - Enquadramento sugerido (close, meio corpo, corpo inteiro)
   - Cenario/fundo ideal
   - Gestos ou movimentos sugeridos
   - Musica/audio sugerido (tendencia atual ou genero)

3. TEXTO PARA LEGENDA (copy do post)
   - Copy com 2-3 linhas + hashtags

4. IDEIAS DE CAPA
   - 2 opcoes de texto para capa do Reels

Formato: texto corrido para teleprompter (uma frase por linha, fonte grande).
```

**Exemplo de resultado esperado:** Roteiro frase por frase com marcacoes de tempo, instrucoes de gravacao, copy da legenda e sugestoes de capa.

### 4. Criador de Headlines para Landing Page

**Quando usar:** Ao construir ou revisar landing pages para o cliente.

**Prompt:**
```
Preciso de headlines matadoras para uma landing page. Crie variacoes testadas para cada secao.

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Produto/servico: [PRODUTO OU SERVICO]
- Publico-alvo: [DESCRICAO]
- Problema principal que resolve: [PROBLEMA]
- Resultado principal que entrega: [RESULTADO]
- Preco: R$ [VALOR] (ou "sob consulta")
- Tipo de LP: [CAPTURA DE LEAD / VENDA DIRETA / AGENDAMENTO / WEBINAR]

Me entregue headlines para cada secao da LP:

1. HEADLINE PRINCIPAL (acima da dobra) — 5 opcoes
   - Cada uma usando abordagem diferente (dor, desejo, curiosidade, numero, pergunta)

2. SUBHEADLINE — 3 opcoes
   - Complementa a headline, adiciona contexto

3. SECAO DE PROBLEMA — 3 opcoes
   - Headlines que nomeiam a dor do publico

4. SECAO DE SOLUCAO — 3 opcoes
   - Headlines que apresentam o produto como resposta

5. SECAO DE BENEFICIOS — 3 opcoes
   - Headlines que destacam o que o cliente ganha

6. SECAO DE PROVA SOCIAL — 3 opcoes
   - Headlines para area de depoimentos/resultados

7. SECAO DE OFERTA — 3 opcoes
   - Headlines para a area de preco/pacotes

8. CTA PRINCIPAL — 5 opcoes
   - Textos para o botao de acao

Total: 28+ headlines organizadas por secao, prontas para testar.
```

**Exemplo de resultado esperado:** Lista de 28+ headlines organizadas por secao da landing page, com variacoes de abordagem para testes A/B.

### 5. Gerador de Email de Follow-Up

**Quando usar:** Quando o cliente precisa de uma sequencia de emails para nutrir leads ou acompanhar propostas.

**Prompt:**
```
Preciso de uma sequencia de emails de follow-up profissional. Crie os emails completos.

CONTEXTO:
- Empresa: [NOME DA EMPRESA]
- Situacao: [LEAD CAPTADO / PROPOSTA ENVIADA / POS-REUNIAO / POS-COMPRA / REATIVACAO]
- Nome do lead/cliente: [NOME] (use como placeholder)
- Produto/servico: [PRODUTO OU SERVICO]
- Acao desejada: [AGENDAR REUNIAO / FECHAR COMPRA / DAR FEEDBACK / RENOVAR]
- Tom: [FORMAL / INFORMAL / CONSULTIVO]
- Quantos emails na sequencia: [3 / 5 / 7]

Para cada email, entregue:

1. ASSUNTO (2 opcoes — uma direta, uma com curiosidade)
2. PRE-HEADER (texto que aparece na preview)
3. CORPO DO EMAIL
   - Saudacao personalizada
   - Contexto (lembrar por que esta recebendo)
   - Conteudo de valor (dica, dado, recurso)
   - CTA claro
   - Assinatura profissional
4. TIMING (quantos dias apos o email anterior)

Regras:
- Cada email deve ter no maximo 150 palavras no corpo
- Nenhum email deve parecer "cobranca" — sempre agregar valor
- O ultimo email da sequencia deve ter tom de "ultima oportunidade" sem ser agressivo
- Inclua variacoes para caso o lead responda ou nao responda
```

**Exemplo de resultado esperado:** Sequencia de 3-7 emails com assunto, corpo e timing definidos, cada um agregando valor e conduzindo para a acao desejada.

### 6. Criador de Posts LinkedIn

**Quando usar:** Quando o cliente tem presenca no LinkedIn e precisa de posts que gerem autoridade e conexoes.

**Prompt:**
```
Preciso criar posts profissionais para LinkedIn. Gere conteudo que funciona nessa plataforma.

INFORMACOES:
- Nome da pessoa/empresa: [NOME]
- Cargo/funcao: [CARGO]
- Segmento: [NICHO]
- Objetivo: [AUTORIDADE / GERAR LEADS / NETWORKING / RECRUTAMENTO]
- Publico no LinkedIn: [QUEM SEGUE — empresarios, RH, gestores, etc.]
- Tema do post: [ASSUNTO ESPECIFICO]

Crie 3 posts com abordagens diferentes:

POST 1 — HISTORIA PESSOAL
- Formato: storytelling com licao de negocio
- Estrutura: gancho provocativo > historia > virada > licao > CTA
- Tamanho: 150-200 palavras

POST 2 — LISTA COM VALOR
- Formato: "X coisas que aprendi sobre [TEMA]"
- Estrutura: gancho > lista numerada > conclusao > CTA
- Tamanho: 150-200 palavras

POST 3 — OPINIAO CONTRARIA
- Formato: perspectiva diferente sobre um tema do setor
- Estrutura: afirmacao ousada > argumentacao > dados/exemplo > CTA
- Tamanho: 150-200 palavras

Para cada post, inclua:
- Texto formatado para LinkedIn (quebras de linha, espacamento)
- Sugestao de imagem/arte para acompanhar
- 3-5 hashtags relevantes
- Melhor horario para publicar

Regras:
- Primeira linha PRECISA ser um gancho forte (aparece no "ver mais")
- Sem jargoes corporativos genericos
- Tom humano, nao robótico
- Nenhum emoji no inicio das linhas (anti-padrao LinkedIn)
```

**Exemplo de resultado esperado:** 3 posts completos formatados para LinkedIn com ganchos fortes, conteudo de valor e CTAs adequados para a plataforma.

### 7. Gerador de Descricao de Servico/Proposta

**Quando usar:** Quando voce precisa descrever seus proprios servicos ou montar uma proposta para o cliente.

**Prompt:**
```
Preciso montar uma descricao profissional de servico para incluir em propostas comerciais.

INFORMACOES:
- Seu nome/empresa: [NOME]
- Servico oferecido: [TRAFEGO PAGO / SOCIAL MEDIA / DESIGN / VIDEO / PACOTE COMPLETO]
- Para quem e: [PERFIL DO CLIENTE IDEAL]
- Valor: R$ [VALOR]/mes ou R$ [VALOR] por projeto
- O que esta incluso: [LISTA DE ENTREGAS]
- Diferenciais: [O QUE VOCE FAZ DE DIFERENTE]
- Cases/resultados anteriores: [DESCREVA 1-2 RESULTADOS]

Me entregue:

1. DESCRICAO DO SERVICO (versao completa — 200 palavras)
   - O que e o servico
   - Para quem e
   - O que esta incluso
   - Qual o resultado esperado

2. DESCRICAO DO SERVICO (versao curta — 50 palavras)
   - Para usar em bio, site ou WhatsApp

3. SECAO "COMO FUNCIONA" (3-5 etapas)
   - Passo a passo simplificado do processo

4. SECAO "POR QUE ESCOLHER [NOME]"
   - 5 bullet points com diferenciais

5. SECAO "RESULTADOS"
   - Formatacao dos cases em formato antes/depois

6. FAQ (5 perguntas frequentes com respostas)
   - Perguntas que clientes fazem antes de fechar

Formate de forma que eu possa copiar direto para uma proposta em PDF ou Google Docs.
```

**Exemplo de resultado esperado:** Proposta completa com descricao de servico, processo, diferenciais, cases formatados e FAQ, pronta para usar em documentos.

### 8. Criador de Carrossel Educativo (10 Slides)

**Quando usar:** Quando o cliente precisa de um carrossel educativo para Instagram e voce precisa entregar texto + briefing visual.

**Prompt:**
```
Preciso criar um carrossel educativo de 10 slides para Instagram. Me entregue o conteudo completo.

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Tema do carrossel: [ASSUNTO]
- Publico-alvo: [QUEM VAI VER]
- Objetivo: [EDUCAR / GERAR SALVAMENTOS / AUTORIDADE / ENGAJAMENTO]
- Tom: [FORMAL / INFORMAL / TECNICO / ACESSIVEL]
- Cores da marca: [COR 1, COR 2, COR 3]

Me entregue SLIDE POR SLIDE:

SLIDE 1 — CAPA
- Titulo chamativo (max 8 palavras)
- Subtitulo (1 linha)
- Instrucao visual para o designer

SLIDES 2-9 — CONTEUDO
Para cada slide:
- Titulo do slide (max 6 palavras)
- Texto principal (max 30 palavras por slide)
- Elemento visual sugerido (icone, grafico, imagem)
- Nota para o designer (layout, destaque, cor de fundo)

SLIDE 10 — CTA
- Frase de fechamento
- CTA (seguir, salvar, compartilhar, clicar no link)
- Instrucao visual

COPY DA LEGENDA:
- Texto do post (resumo do conteudo + CTA + hashtags)

Regras:
- Pouco texto por slide (ninguem le textao em carrossel)
- Cada slide deve fazer sentido sozinho
- Hierarquia visual clara (titulo > destaque > texto de apoio)
- O slide 1 precisa gerar curiosidade suficiente pra pessoa deslizar
```

**Exemplo de resultado esperado:** 10 slides com textos, instrucoes visuais para designer e copy da legenda, prontos para produzir.

### 9. Gerador de Bio Profissional

**Quando usar:** Ao configurar ou otimizar perfis de redes sociais do cliente.

**Prompt:**
```
Preciso criar bios profissionais otimizadas para redes sociais. Gere versoes para cada plataforma.

INFORMACOES:
- Nome da empresa/pessoa: [NOME]
- Segmento: [NICHO]
- Produto/servico principal: [O QUE VENDE]
- Diferencial em uma frase: [O QUE FAZ DE DIFERENTE]
- Publico-alvo: [PARA QUEM]
- Cidade: [LOCAL]
- Link principal: [URL DO SITE / LINKTREE / WHATSAPP]
- Tem prova social? [NUMERO DE CLIENTES, ANOS DE MERCADO, PREMIO, ETC.]

Me entregue bios para:

1. INSTAGRAM (max 150 caracteres)
   - 3 versoes diferentes
   - Com emojis estrategicos
   - Com CTA claro

2. LINKEDIN (headline + resumo)
   - Headline (max 120 caracteres) — 3 versoes
   - Resumo (max 200 palavras) — 1 versao completa

3. TIKTOK (max 80 caracteres)
   - 3 versoes diferentes

4. WHATSAPP BUSINESS (descricao)
   - 1 versao completa
   - Horario de atendimento sugerido

5. GOOGLE MEU NEGOCIO (se aplicavel)
   - Descricao otimizada para SEO local (max 750 caracteres)

Para cada bio, siga as regras:
- Primeira linha = o que faz (clareza)
- Segunda linha = pra quem / diferencial
- Terceira linha = CTA ou prova social
- Use palavras-chave do nicho para buscas
```

**Exemplo de resultado esperado:** Bios otimizadas para 5 plataformas diferentes, com variacoes para teste e formatacao especifica de cada rede.

### 10. Criador de CTA por Contexto

**Quando usar:** Quando precisa de CTAs variados para diferentes pecas e contextos de comunicacao.

**Prompt:**
```
Preciso de CTAs (chamadas para acao) poderosos para diferentes contextos. Gere opcoes variadas.

INFORMACOES:
- Empresa: [NOME DA EMPRESA]
- Produto/servico: [PRODUTO OU SERVICO]
- Acao desejada: [COMPRAR / AGENDAR / CADASTRAR / BAIXAR / SEGUIR / ENVIAR MENSAGEM]
- Tom da marca: [FORMAL / INFORMAL / URGENTE / CONSULTIVO]

Me entregue CTAs para cada contexto:

1. ANUNCIOS (Meta/Google) — 10 opcoes
   - Botao do anuncio
   - Texto que antecede o CTA na copy

2. LANDING PAGE — 10 opcoes
   - Botao principal
   - Botao secundario
   - Texto que antecede o botao

3. POSTS INSTAGRAM — 10 opcoes
   - CTA para legendas (feed)
   - CTA para stories
   - CTA para reels

4. EMAILS — 10 opcoes
   - CTA no corpo do email
   - CTA no PS

5. WHATSAPP — 10 opcoes
   - CTA para mensagens de follow-up
   - CTA para broadcast

Para cada CTA, classifique:
- Nivel de urgencia (baixo/medio/alto)
- Melhor momento para usar (topo/meio/fundo de funil)

Total: 50 CTAs organizados por contexto e nivel de urgencia.
```

**Exemplo de resultado esperado:** Banco de 50 CTAs organizados por plataforma, com classificacao de urgencia e momento do funil ideal para uso.
