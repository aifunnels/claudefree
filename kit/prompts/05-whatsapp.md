# Categoria 5: WhatsApp e Atendimento

> Prompts para comunicacao profissional com clientes via WhatsApp e email.

---

### 1. Mensagem de Boas-Vindas para Cliente Novo

**Quando usar:** Logo apos fechar contrato com um cliente novo, para causar uma otima primeira impressao.

**Prompt:**
```
Preciso de uma mensagem de boas-vindas profissional para enviar a um cliente novo via WhatsApp.

INFORMACOES:
- Nome do cliente: [NOME]
- Nome da empresa do cliente: [EMPRESA]
- Servico contratado: [TRAFEGO PAGO / SOCIAL MEDIA / DESIGN / VIDEO / COMPLETO]
- Seu nome: [SEU NOME]
- Sua empresa: [SUA EMPRESA]
- Data de inicio do trabalho: [DATA]
- Primeiro passo que o cliente precisa fazer: [ENVIAR ACESSOS / PREENCHER BRIEFING / AGENDAR REUNIAO]

Crie 3 versoes de mensagem de boas-vindas:

VERSAO 1 — PROFISSIONAL E DIRETA
- Agradecimento pela confianca
- Resumo do que foi contratado
- Proximos passos numerados
- Prazo do primeiro entregavel
- Canal de comunicacao e horario de atendimento

VERSAO 2 — CALOROSA E ENTUSIASMADA
- Tom mais proximo, mostrando empolgacao
- Mesmas informacoes da versao 1, com mais personalidade

VERSAO 3 — FORMAL (para clientes corporativos)
- Tom institucional
- Mesmas informacoes, linguagem mais corporativa

Para cada versao, inclua:
- Mensagem principal (max 15 linhas para WhatsApp)
- Mensagem complementar com checklist do que o cliente precisa enviar
- Template de resposta caso o cliente demore pra responder (follow-up em 24h)

Regras:
- Funcionar bem no WhatsApp (sem formatacao complexa)
- Transmitir profissionalismo e organizacao
- Deixar claro o que o cliente precisa fazer
- Nao ser mensagem generica — parecer personalizada
```

**Exemplo de resultado esperado:** 3 versoes de mensagem de boas-vindas com tons diferentes, checklist complementar e template de follow-up.

### 2. Follow-Up Pos-Reuniao

**Quando usar:** Depois de qualquer reuniao com cliente, para documentar o que foi alinhado e os proximos passos.

**Prompt:**
```
Preciso de uma mensagem de follow-up profissional para enviar apos uma reuniao com cliente.

INFORMACOES:
- Nome do cliente: [NOME]
- Tipo de reuniao: [KICKOFF / ALINHAMENTO MENSAL / REVIEW / ESTRATEGIA / COBRANCA]
- Data da reuniao: [DATA]
- Principais pontos discutidos: [LISTE OS TOPICOS]
- Decisoes tomadas: [LISTE AS DECISOES]
- Proximos passos combinados: [LISTE COM RESPONSAVEIS]
- Prazos definidos: [LISTE OS PRAZOS]

Crie a mensagem de follow-up com:

1. AGRADECIMENTO pela reuniao (1 linha)

2. RESUMO DO QUE FOI ALINHADO
   - Pontos principais em bullets
   - Decisoes tomadas

3. PROXIMOS PASSOS (formato checklist)
   - O que VOCE vai fazer + prazo
   - O que O CLIENTE precisa fazer + prazo

4. PENDENCIAS (se houver)
   - Itens que ficaram em aberto
   - Quem resolve e ate quando

5. ENCERRAMENTO
   - Disponibilidade para duvidas
   - Proximo ponto de contato (data da proxima reuniao ou entrega)

Crie 2 versoes:
- VERSAO WHATSAPP (curta, max 20 linhas)
- VERSAO EMAIL (mais detalhada, formato profissional)

Regras:
- Ser especifico (nada generico)
- Demonstrar organizacao e controle
- Documentar para evitar "isso nao foi combinado" no futuro
```

**Exemplo de resultado esperado:** Mensagem de follow-up em 2 versoes (WhatsApp e email) com resumo, decisoes, proximos passos e prazos.

### 3. Lembrete de Deadline

**Quando usar:** Quando o cliente precisa enviar algo e o prazo esta chegando (ou ja passou).

**Prompt:**
```
Preciso enviar um lembrete educado mas firme sobre um prazo para o cliente.

INFORMACOES:
- Nome do cliente: [NOME]
- O que o cliente precisa enviar/fazer: [DESCRICAO]
- Prazo original: [DATA]
- Status: [PRAZO CHEGANDO / PRAZO HOJE / PRAZO VENCIDO HA X DIAS]
- Impacto do atraso: [O QUE ACONTECE SE NAO ENVIAR — ex: campanha atrasa, post nao sai, etc.]
- E a primeira cobranca ou ja cobrou antes? [PRIMEIRA / SEGUNDA / TERCEIRA]

Crie a mensagem adequada para cada situacao:

SE PRAZO CHEGANDO (2-3 dias antes):
- Tom: amigavel, lembrete gentil
- Reforcar o que precisa e ate quando
- Oferecer ajuda se tiver dificuldade

SE PRAZO E HOJE:
- Tom: direto, com leve urgencia
- Reforcar a importancia
- Perguntar se consegue enviar ate o fim do dia

SE PRAZO VENCIDO (primeira cobranca):
- Tom: profissional, sem ser agressivo
- Informar que o prazo passou
- Explicar o impacto do atraso no cronograma
- Pedir nova data

SE PRAZO VENCIDO (segunda/terceira cobranca):
- Tom: firme mas respeitoso
- Reiterar impacto concreto no trabalho
- Pedir posicao definitiva
- Sugerir alternativa se o cliente nao conseguir cumprir

Regras:
- NUNCA ser passivo-agressivo
- NUNCA culpar o cliente explicitamente
- Sempre manter o tom profissional
- Sempre oferecer solucao/alternativa
- Max 10 linhas cada mensagem
```

**Exemplo de resultado esperado:** 4 mensagens para diferentes estagios de atraso, com tom adequado e alternativas.

### 4. Resposta a Reclamacao de Cliente

**Quando usar:** Quando o cliente reclama de algo (resultado, prazo, qualidade) e voce precisa responder profissionalmente.

**Prompt:**
```
Preciso responder a uma reclamacao de cliente de forma profissional e estrategica.

RECLAMACAO DO CLIENTE:
"""
[COLE A MENSAGEM DO CLIENTE AQUI]
"""

CONTEXTO:
- Nome do cliente: [NOME]
- Tempo de contrato: [MESES]
- A reclamacao e justificada? [SIM / PARCIALMENTE / NAO]
- O que realmente aconteceu: [SUA VERSAO]
- O que voce pode fazer para resolver: [OPCOES]
- Nivel de risco de perder o cliente: [ALTO / MEDIO / BAIXO]

Crie a resposta seguindo esta estrutura:

1. ACOLHIMENTO (nao se defender primeiro — ouvir)
   - Agradecer o feedback
   - Mostrar que entendeu a frustracão
   - Nao minimizar o sentimento do cliente

2. CONTEXTUALIZACAO (sem desculpas excessivas)
   - Explicar o que aconteceu de forma transparente
   - Assumir responsabilidade pelo que for justo
   - Corrigir expectativas se necessario (sem arrogancia)

3. SOLUCAO CONCRETA
   - O que voce vai fazer para resolver
   - Prazo para a solucao
   - O que vai mudar para nao acontecer de novo

4. REAFIRMACAO DE COMPROMISSO
   - Reforcar o valor que ja entregou
   - Demonstrar comprometimento com o resultado
   - Sugerir uma conversa se o assunto for complexo

Crie 2 versoes:
- VERSAO RAPIDA (WhatsApp — max 10 linhas, para resposta imediata)
- VERSAO COMPLETA (email ou mensagem longa — mais detalhada)

Regras:
- NUNCA ser defensivo ou atacar o cliente
- NUNCA prometer o que nao pode cumprir
- Responder em ate 2 horas (a versao rapida)
- Tom: seguro, empático, solucionador
```

**Exemplo de resultado esperado:** 2 versoes de resposta (rapida e completa) com acolhimento, contextualizacao, solucao concreta e reafirmacao.

### 5. Pedido de Feedback/Depoimento

**Quando usar:** Quando quiser coletar depoimentos de clientes satisfeitos para usar como prova social.

**Prompt:**
```
Preciso pedir feedback e depoimento para um cliente de forma natural, sem parecer forcado.

INFORMACOES:
- Nome do cliente: [NOME]
- Tempo de contrato: [MESES]
- Resultado relevante entregue: [DESCREVA O RESULTADO — ex: "triplicamos os leads", "crescemos 5k seguidores"]
- Onde vai usar o depoimento: [INSTAGRAM / SITE / PROPOSTA / GOOGLE]
- Formato desejado: [TEXTO / VIDEO / AUDIO / AVALIACAO GOOGLE]

Crie 3 abordagens diferentes:

ABORDAGEM 1 — DIRETA (pos-resultado)
- Mencionar o resultado alcancado
- Pedir o depoimento de forma objetiva
- Facilitar a vida do cliente (dar roteiro ou perguntas guia)

ABORDAGEM 2 — INDIRETA (embutida em conversa)
- Perguntar "como esta se sentindo com os resultados?"
- Usar a resposta positiva como gancho para pedir o depoimento

ABORDAGEM 3 — COM INCENTIVO
- Oferecer algo em troca (ex: destaque no perfil, desconto, material extra)
- Sem parecer "compra" de depoimento

Para cada abordagem, inclua:
- Mensagem principal
- Perguntas guia para o cliente responder (facilitar o depoimento)
  1. Qual era sua situacao antes?
  2. O que mudou depois?
  3. Qual resultado mais te surpreendeu?
  4. Recomendaria para outros empresarios?
- Template de agradecimento apos receber o depoimento

Regras:
- Nao pressionar — dar abertura para o cliente recusar
- Facilitar ao maximo (quanto menos esforco pro cliente, melhor)
- Tom: genuino, nao "marketeiro"
```

**Exemplo de resultado esperado:** 3 mensagens com abordagens diferentes, perguntas guia para facilitar o depoimento e template de agradecimento.

### 6. Apresentacao de Resultados via WhatsApp

**Quando usar:** Para mandar um update rapido de resultados pelo WhatsApp (entre relatorios mensais completos).

**Prompt:**
```
Preciso enviar resultados de performance para o cliente via WhatsApp de forma impactante e rapida.

INFORMACOES:
- Nome do cliente: [NOME]
- Periodo: [SEMANA / QUINZENA / DESTAQUE DO DIA]
- Servico: [TRAFEGO / SOCIAL MEDIA / COMPLETO]

DADOS:
[LISTE AS METRICAS QUE QUER COMPARTILHAR]

DESTAQUE PRINCIPAL:
[O RESULTADO MAIS IMPRESSIONANTE DO PERIODO]

Crie uma mensagem de resultados que:

1. ABRA COM IMPACTO
   - O numero mais impressionante primeiro
   - Gerar a sensacao de "valeu a pena investir"

2. CONTEXTUALIZE
   - Comparar com periodo anterior ou meta
   - "Isso significa que..." (traduzir numero em beneficio)

3. MOSTRE OS DADOS (formato limpo)
   - Metricas organizadas
   - Sem excesso — so o que importa

4. DESTAQUE 1-2 APRENDIZADOS
   - O que esta funcionando e vai ser escalado
   - Algum teste que deu certo

5. FECHE COM CONFIANCA
   - Proximo passo
   - "Estamos no caminho certo" (se estiver)

Crie 2 versoes:
- VERSAO CURTA (max 8 linhas — para update rapido)
- VERSAO MEDIA (max 15 linhas — para update mais completo)

Regras:
- Formatacao que funcione no WhatsApp
- Numeros em destaque
- Tom: confiante sem ser arrogante
- Se os resultados NAO forem bons, adapte o tom para "transparente e com plano de acao"
```

**Exemplo de resultado esperado:** 2 versoes de update de resultados formatadas para WhatsApp, com destaque nos numeros e contextualizacao.

### 7. Proposta de Upsell/Upgrade

**Quando usar:** Quando identificar oportunidade de oferecer servicos adicionais a um cliente existente.

**Prompt:**
```
Preciso enviar uma proposta de upsell para um cliente existente de forma natural e consultiva.

INFORMACOES:
- Nome do cliente: [NOME]
- Servico atual: [O QUE JA FAZ PARA ELE]
- Valor atual: R$ [VALOR]/mes
- Resultado entregue: [RESULTADO QUE JUSTIFICA O UPSELL]
- Servico adicional proposto: [O QUE QUER OFERECER]
- Valor do adicional: R$ [VALOR]
- Por que faz sentido agora: [JUSTIFICATIVA BASEADA EM DADOS]

Crie a mensagem de upsell em 3 etapas:

MENSAGEM 1 — PREPARACAO (enviar 2-3 dias antes da proposta)
- Compartilhar um resultado positivo
- Plantar a semente: "Isso me fez pensar em uma oportunidade..."
- Nao vender ainda — so gerar curiosidade

MENSAGEM 2 — PROPOSTA (enviar apos resposta positiva)
- Referenciar o resultado
- Apresentar o servico adicional como consequencia logica
- Mostrar o que muda nos resultados com o upgrade
- Valor e condicoes
- Sem pressao — tom consultivo

MENSAGEM 3 — FOLLOW-UP (se nao responder em 48h)
- Reforcar o beneficio principal
- Criar leve urgencia (sem ser falsa)
- Facilitar a decisao: "Posso te explicar melhor numa call de 15min?"

Regras:
- NUNCA parecer que esta empurrando servico
- Base SEMPRE em dados e resultados reais
- Tom: consultor que quer ajudar, nao vendedor
- O cliente precisa sentir que e a decisao DELE
- Max 10 linhas cada mensagem
```

**Exemplo de resultado esperado:** Sequencia de 3 mensagens que conduz naturalmente para o upsell, baseada em dados e com tom consultivo.

### 8. Mensagem de Renovacao de Contrato

**Quando usar:** Quando o contrato do cliente esta vencendo e voce precisa iniciar a conversa de renovacao.

**Prompt:**
```
Preciso enviar uma mensagem para iniciar a conversa de renovacao de contrato com o cliente.

INFORMACOES:
- Nome do cliente: [NOME]
- Tempo de contrato: [MESES]
- Vencimento: [DATA]
- Resultado principal entregue: [RESULTADO MAIS RELEVANTE]
- Valor atual: R$ [VALOR]/mes
- Quer propor reajuste? [SIM — novo valor R$ X / NAO]
- Nivel de satisfacao percebido: [ALTO / MEDIO / BAIXO]
- Risco de churn: [ALTO / MEDIO / BAIXO]

Crie mensagens para cada cenario:

CENARIO 1 — CLIENTE SATISFEITO (baixo risco)
- Agradecer a parceria
- Resumir os resultados do periodo
- Propor renovacao com condicoes
- Tom: parceria de longo prazo

CENARIO 2 — CLIENTE NEUTRO (medio risco)
- Agradecer a parceria
- Reconhecer pontos de melhoria
- Apresentar plano para o proximo ciclo
- Oferecer condicao especial de renovacao
- Tom: comprometimento com evolucao

CENARIO 3 — CLIENTE INSATISFEITO (alto risco)
- Reconhecer que houve desafios
- Mostrar o que foi aprendido
- Apresentar plano de acao concreto para o proximo ciclo
- Oferecer periodo de "teste" ou condicao diferenciada
- Tom: transparencia e compromisso

Para cada cenario:
- Mensagem via WhatsApp (max 15 linhas)
- Email formal com detalhes da renovacao
- Follow-up caso nao responda em 3 dias

Regras:
- Iniciar a conversa 15-30 dias antes do vencimento
- NUNCA dar ultimato ou pressionar
- Sempre ter dados de resultado na mensagem
- Se propor reajuste, justificar com base em valor entregue
```

**Exemplo de resultado esperado:** 3 cenarios com mensagens adaptadas ao nivel de satisfacao, versoes WhatsApp e email, e follow-ups.

### 9. Aviso de Atraso na Entrega

**Quando usar:** Quando voce vai atrasar uma entrega e precisa comunicar ao cliente antes do prazo.

**Prompt:**
```
Preciso comunicar ao cliente que uma entrega vai atrasar, de forma profissional e proativa.

INFORMACOES:
- Nome do cliente: [NOME]
- Entrega que vai atrasar: [DESCRICAO]
- Prazo original: [DATA]
- Novo prazo previsto: [DATA]
- Motivo real do atraso: [MOTIVO]
- E a primeira vez que atrasa com esse cliente? [SIM/NAO]
- Impacto no projeto geral: [BAIXO / MEDIO / ALTO]

Crie a mensagem seguindo esta estrutura:

1. COMUNICAR PROATIVAMENTE (antes do prazo vencer, nao depois)
   - Ir direto ao ponto
   - Informar o que vai atrasar e o novo prazo

2. EXPLICAR SEM DAR DESCULPA
   - Motivo objetivo (sem drama ou vitimismo)
   - Nao culpar terceiros
   - Assumir responsabilidade

3. APRESENTAR SOLUCAO
   - Novo prazo realista (com margem)
   - O que voce esta fazendo para resolver
   - Se possivel, entregar algo parcial antes

4. COMPENSAR (se atraso for significativo)
   - Oferecer algo extra como compensacao
   - Demonstrar que se importa com o prazo do cliente

5. REAFIRMAR COMPROMISSO
   - Garantir que o novo prazo sera cumprido
   - Se colocar a disposicao para conversar

Crie 2 versoes:
- VERSAO LEVE (atraso de 1-2 dias, baixo impacto)
- VERSAO COMPLETA (atraso significativo, medio/alto impacto)

Regras:
- NUNCA avisar o atraso no dia do prazo ou depois
- NUNCA mentir sobre o motivo
- NUNCA prometer novo prazo que nao vai cumprir
- Tom: transparente, responsavel, solucionador
- Max 10 linhas cada versao
```

**Exemplo de resultado esperado:** 2 versoes de comunicado de atraso, uma leve e uma para situacoes mais serias, com solucao e compensacao.

### 10. Encerramento de Contrato Profissional

**Quando usar:** Quando o contrato com um cliente chega ao fim (por decisao sua ou dele) e voce precisa encerrar profissionalmente.

**Prompt:**
```
Preciso encerrar um contrato com um cliente de forma profissional, mantendo a porta aberta para o futuro.

INFORMACOES:
- Nome do cliente: [NOME]
- Tempo de contrato: [MESES]
- Motivo do encerramento: [DECISAO DO CLIENTE / DECISAO SUA / MUTUO / FIM DO PROJETO]
- Resultado entregue durante o contrato: [RESUMO]
- Ultimo dia de trabalho: [DATA]
- Ha pendencias? [SIM/NAO — quais]
- Quer manter relacionamento para o futuro? [SIM/NAO]

Crie mensagens para cada cenario:

CENARIO 1 — CLIENTE PEDIU PRA SAIR
- Respeitar a decisao sem questionar
- Agradecer a parceria
- Oferecer transicao organizada
- Deixar porta aberta

CENARIO 2 — VOCE DECIDIU SAIR
- Comunicar com antecedencia (minimo 30 dias)
- Explicar o motivo de forma profissional
- Oferecer suporte na transicao
- Indicar outro profissional (se possivel)

CENARIO 3 — FIM NATURAL DO PROJETO
- Celebrar o que foi conquistado
- Resumir os resultados
- Apresentar recomendacoes para continuidade
- Oferecer retainer ou consultoria pontual

Para cada cenario, inclua:

1. MENSAGEM DE COMUNICACAO INICIAL (WhatsApp)
2. EMAIL FORMAL DE ENCERRAMENTO com:
   - Agradecimento
   - Resumo de resultados
   - Entregas finais e pendencias
   - Transferencia de acessos e materiais
   - Recomendacoes para continuidade
   - Disponibilidade futura
3. CHECKLIST DE ENCERRAMENTO
   - Acessos a devolver
   - Arquivos a entregar
   - Faturas pendentes
   - Documentacao de senhas/acessos

Regras:
- NUNCA queimar pontes, mesmo que o cliente tenha sido dificil
- SEMPRE entregar tudo que e do cliente (acessos, arquivos, dados)
- Profissionalismo ate o ultimo dia
- Tom: grato, maduro, disponivel
```

**Exemplo de resultado esperado:** Mensagens de encerramento para 3 cenarios, email formal, checklist de transicao e recomendacoes para continuidade.
