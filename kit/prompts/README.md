## Como Usar

1. Escolha o prompt adequado para a situacao
2. Copie o prompt completo
3. Substitua os placeholders entre colchetes [ASSIM] pelas informacoes reais
4. Cole no Claude, ChatGPT ou outra IA de sua preferencia
5. Revise o resultado e adapte se necessario

## Dicas

- Quanto mais informacao voce preencher nos placeholders, melhor o resultado
- Use os prompts de REVISAO para verificar os outputs dos prompts de CRIACAO
- Combine prompts: use o de BRIEFING no inicio, CRIACAO para produzir e REVISAO antes de entregar
- Salve os resultados bons como templates para reutilizar com outros clientes
