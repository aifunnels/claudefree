# Categoria 4: Relatorio

> Prompts para montar relatorios e analises de performance para clientes.

---

### 1. Montador de Relatorio Mensal de Trafego

**Quando usar:** No final de cada mes para montar o relatorio de performance de campanhas pagas.

**Prompt:**
```
Voce e um analista de trafego pago. Preciso que voce monte um relatorio mensal profissional com os dados que vou fornecer.

DADOS DO MES:
- Cliente: [NOME DA EMPRESA]
- Periodo: [MES/ANO]
- Plataforma: [META ADS / GOOGLE ADS / AMBOS]
- Orcamento investido: R$ [VALOR]
- Objetivo da campanha: [LEADS / VENDAS / TRAFEGO / CADASTROS]

METRICAS PRINCIPAIS:
- Impressoes: [NUMERO]
- Alcance: [NUMERO]
- Cliques: [NUMERO]
- CTR: [PERCENTUAL]
- CPC: R$ [VALOR]
- Leads gerados: [NUMERO]
- CPL: R$ [VALOR]
- Vendas: [NUMERO]
- CPA: R$ [VALOR]
- ROAS: [VALOR]

METRICAS DO MES ANTERIOR (para comparacao):
- [REPITA AS METRICAS DO MES ANTERIOR]

Monte o relatorio com a seguinte estrutura:

1. RESUMO EXECUTIVO (3-4 linhas)
   - Resultado principal do mes em linguagem simples
   - Destaque positivo e ponto de atencao

2. DASHBOARD DE METRICAS
   - Tabela com metricas principais + comparativo mes anterior + variacao percentual
   - Setas indicando se subiu ou desceu

3. ANALISE DE PERFORMANCE
   - O que funcionou bem e por que
   - O que nao performou e por que
   - Fatores externos que influenciaram (sazonalidade, mercado)

4. CAMPANHAS DETALHADAS
   - Performance por campanha (se houver mais de uma)
   - Top 3 anuncios que mais performaram
   - Top 3 publicos que mais performaram

5. RECOMENDACOES PARA O PROXIMO MES
   - 3-5 acoes especificas com justificativa
   - Sugestao de orcamento
   - Novos testes propostos

6. PROXIMOS PASSOS
   - O que sera feito na primeira semana do proximo mes
   - Prazos

Formate de maneira visual e facil de entender para um cliente nao tecnico.
```

**Exemplo de resultado esperado:** Relatorio completo de 2-3 paginas com resumo executivo, tabela de metricas, analise detalhada e recomendacoes acionaveis.

### 2. Analise de Performance de Criativos

**Quando usar:** Para analisar quais criativos estao performando melhor e gerar insights para novos testes.

**Prompt:**
```
Preciso analisar a performance dos criativos de campanha e gerar insights acionaveis.

DADOS DOS CRIATIVOS:
- Cliente: [NOME DA EMPRESA]
- Periodo: [PERIODO]
- Plataforma: [META ADS / GOOGLE ADS / TIKTOK ADS]

CRIATIVO 1:
- Nome/ID: [IDENTIFICACAO]
- Formato: [IMAGEM / VIDEO / CARROSSEL]
- Angulo de copy: [DOR / DESEJO / PROVA SOCIAL / CURIOSIDADE]
- Impressoes: [NUMERO]
- Cliques: [NUMERO]
- CTR: [PERCENTUAL]
- Conversoes: [NUMERO]
- CPA: R$ [VALOR]
- Valor gasto: R$ [VALOR]

[REPITA PARA CADA CRIATIVO — minimo 3, ideal 5-10]

Me entregue:

1. RANKING DE CRIATIVOS
   - Do melhor para o pior (por CPA ou CTR, conforme objetivo)
   - Tabela comparativa

2. ANALISE DE PADROES
   - Qual formato performa melhor? (imagem vs video vs carrossel)
   - Qual angulo de copy performa melhor? (dor vs desejo vs prova social)
   - Qual estilo visual performa melhor?
   - Qual duracao de video performa melhor? (se aplicavel)

3. INSIGHTS PARA NOVOS CRIATIVOS
   - 5 hipoteses baseadas nos dados
   - 3 ideias de novos criativos para testar
   - O que PARAR de fazer (criativos que nao performam)

4. PLANO DE TESTES
   - Proximo teste A/B sugerido
   - Variavel a ser testada
   - Orcamento sugerido para o teste
   - Tempo necessario para resultado significativo

Formate como um report visual com tabelas e destaques.
```

**Exemplo de resultado esperado:** Ranking de criativos, padroes identificados, 3 ideias de novos criativos e plano de testes A/B.

### 3. Relatorio de Engajamento Social Media

**Quando usar:** No final do mes para reportar resultados de gestao de redes sociais ao cliente.

**Prompt:**
```
Preciso montar um relatorio de engajamento de redes sociais para o cliente.

DADOS:
- Cliente: [NOME DA EMPRESA]
- Periodo: [MES/ANO]
- Rede principal: [INSTAGRAM / TIKTOK / LINKEDIN]

METRICAS DO MES:
- Seguidores inicio do mes: [NUMERO]
- Seguidores fim do mes: [NUMERO]
- Posts publicados: [NUMERO]
- Reels publicados: [NUMERO]
- Stories publicados: [NUMERO]
- Alcance total: [NUMERO]
- Impressoes totais: [NUMERO]
- Engajamento total (curtidas + comentarios + salvamentos + compartilhamentos): [NUMERO]
- Taxa de engajamento: [PERCENTUAL]
- Visitas ao perfil: [NUMERO]
- Cliques no link: [NUMERO]

TOP 3 POSTS DO MES:
- Post 1: [DESCRICAO + METRICAS]
- Post 2: [DESCRICAO + METRICAS]
- Post 3: [DESCRICAO + METRICAS]

Monte o relatorio com:

1. RESUMO DO MES (3-4 linhas)
   - Destaque principal
   - Crescimento vs mes anterior

2. METRICAS EM DESTAQUE
   - Tabela com metricas + mes anterior + variacao
   - Destaques visuais nos numeros mais relevantes

3. TOP CONTEUDOS
   - Analise dos 3 melhores posts (por que performaram)
   - Analise do pior post (por que nao foi bem)
   - Padroes identificados

4. ANALISE POR FORMATO
   - Feed vs Reels vs Stories vs Carrossel
   - Qual formato gera mais alcance
   - Qual formato gera mais engajamento
   - Qual formato gera mais cliques

5. CRESCIMENTO DE COMUNIDADE
   - Evolucao de seguidores
   - Perfil dos novos seguidores (se disponivel)
   - Taxa de retencao

6. RECOMENDACOES
   - O que manter na estrategia
   - O que mudar
   - 3 ideias de conteudo para o proximo mes
   - Melhor dia/horario para postar (baseado nos dados)

Linguagem acessivel para o cliente entender sem ser especialista.
```

**Exemplo de resultado esperado:** Relatorio completo com metricas, analise de top posts, comparativo de formatos e recomendacoes para o proximo mes.

### 4. Dashboard de KPIs em Texto

**Quando usar:** Para updates rapidos semanais ou quinzenais, quando nao precisa de um relatorio completo.

**Prompt:**
```
Preciso criar um dashboard em texto, rapido e visual, para enviar como update semanal ao cliente.

DADOS:
- Cliente: [NOME DA EMPRESA]
- Periodo: [SEMANA/QUINZENA]
- Tipo de servico: [TRAFEGO / SOCIAL MEDIA / COMPLETO]

METRICAS DA SEMANA:
[LISTE AS METRICAS QUE TEM DISPONIVEIS]

Me entregue um DASHBOARD EM TEXTO formatado para enviar por WhatsApp ou email com:

1. HEADER
   - Nome do cliente + periodo + tipo de report

2. METRICAS CHAVE (formato visual com emojis/simbolos)
   - Cada metrica com: nome | valor atual | meta | status (acima/abaixo da meta)
   - Use cores em texto: VERDE (acima da meta), AMARELO (proximo), VERMELHO (abaixo)

3. DESTAQUES DA SEMANA (max 3)
   - O que mais chamou atencao positivamente

4. PONTOS DE ATENCAO (max 3)
   - O que precisa de ajuste

5. ACOES DA PROXIMA SEMANA (max 3)
   - O que sera feito

Regras:
- MAXIMO 20 linhas totais
- Formatacao que funcione no WhatsApp (sem markdown complexo)
- Numeros em destaque
- Linguagem direta, sem enrolacao
```

**Exemplo de resultado esperado:** Dashboard compacto de 15-20 linhas formatado para WhatsApp com metricas, destaques e acoes.

### 5. Analise Comparativa Mes a Mes

**Quando usar:** Para comparar performance entre dois ou mais meses e identificar tendencias.

**Prompt:**
```
Preciso de uma analise comparativa entre meses para identificar tendencias e ajustar estrategia.

DADOS:
- Cliente: [NOME DA EMPRESA]
- Servico: [TRAFEGO / SOCIAL MEDIA / COMPLETO]
- Periodos comparados: [MES 1] vs [MES 2] (adicione mais meses se tiver)

METRICAS MES 1 ([MES]):
[LISTE TODAS AS METRICAS]

METRICAS MES 2 ([MES]):
[LISTE TODAS AS METRICAS]

Me entregue:

1. TABELA COMPARATIVA
   - Todas as metricas lado a lado
   - Variacao percentual entre meses
   - Indicador de melhora/piora

2. ANALISE DE TENDENCIAS
   - Metricas que estao MELHORANDO consistentemente
   - Metricas que estao PIORANDO consistentemente
   - Metricas ESTAVEIS

3. CORRELACOES
   - Mudancas que podem estar conectadas (ex: mais investimento = mais leads?)
   - Hipoteses sobre o que causou as variacoes

4. PROJECAO
   - Se a tendencia se mantiver, qual o cenario para o proximo mes?
   - Cenario otimista vs pessimista vs realista

5. ACOES BASEADAS NOS DADOS
   - O que ESCALAR (esta funcionando e pode crescer)
   - O que OTIMIZAR (tem potencial mas precisa ajuste)
   - O que CORTAR (nao esta funcionando)

6. META PARA PROXIMO MES
   - KPIs com metas numericas baseadas nos dados

Formate de forma visual com tabela e destaques.
```

**Exemplo de resultado esperado:** Tabela comparativa, analise de tendencias, projecoes e metas numericas para o proximo mes.

### 6. Relatorio de Onboarding (Primeiros 30 Dias)

**Quando usar:** Ao completar o primeiro mes com um cliente novo, para mostrar o que foi feito e alinhar os proximos passos.

**Prompt:**
```
Preciso montar um relatorio dos primeiros 30 dias de trabalho com um cliente novo.

INFORMACOES:
- Cliente: [NOME DA EMPRESA]
- Servico contratado: [TIPO]
- Data de inicio: [DATA]
- Orcamento mensal: R$ [VALOR]
- Objetivo combinado: [OBJETIVO]
- KPIs definidos: [LISTA]

O QUE FOI FEITO NOS 30 DIAS:
[LISTE TUDO QUE FOI IMPLEMENTADO — setup de contas, campanhas, conteudos, ajustes, etc.]

RESULTADOS PRELIMINARES:
[METRICAS QUE TEM DISPONIVEL]

Monte o relatorio com:

1. RESUMO EXECUTIVO
   - O que foi entregue nos primeiros 30 dias
   - Primeiro resultado tangivel
   - Status geral: dentro do esperado / acima / abaixo

2. TIMELINE DO QUE FOI FEITO
   - Semana 1: [O que foi feito]
   - Semana 2: [O que foi feito]
   - Semana 3: [O que foi feito]
   - Semana 4: [O que foi feito]

3. ENTREGAS REALIZADAS
   - Lista completa com status (feito/em andamento/pendente)

4. RESULTADOS INICIAIS
   - Metricas disponíveis com contexto
   - Observacao: "Primeiros 30 dias sao de setup e aprendizado — resultados consistentes a partir do mes 2-3"

5. APRENDIZADOS
   - O que descobrimos sobre o publico
   - O que funciona para esse nicho
   - Ajustes feitos durante o mes

6. PLANO PARA OS PROXIMOS 60 DIAS
   - Mes 2: foco em [OTIMIZACAO/ESCALA/TESTE]
   - Mes 3: foco em [ESCALA/CONSOLIDACAO]
   - Metas numericas para cada mes

7. ALINHAMENTOS NECESSARIOS
   - Decisoes que dependem do cliente
   - Materiais necessarios
   - Prazos

Formate de forma profissional — este relatorio define a relacao para os proximos meses.
```

**Exemplo de resultado esperado:** Relatorio de onboarding com timeline, entregas, resultados iniciais e plano para os meses 2 e 3.

### 7. Resumo Executivo para Cliente

**Quando usar:** Quando o cliente nao tem tempo de ler o relatorio completo e precisa de um resumo rapido.

**Prompt:**
```
Preciso transformar um relatorio detalhado em um resumo executivo curto e direto para o cliente.

RELATORIO COMPLETO:
"""
[COLE O RELATORIO AQUI]
"""

PERFIL DO CLIENTE:
- Nome: [NOME]
- Nivel de entendimento tecnico: [BASICO / INTERMEDIARIO / AVANCADO]
- O que mais importa pra ele: [VENDAS / LEADS / SEGUIDORES / BRAND]
- Prefere: [NUMEROS / NARRATIVA / AMBOS]

Me entregue um RESUMO EXECUTIVO com:

1. RESULTADO PRINCIPAL (1 frase)
   - O numero mais importante do periodo

2. O QUE DEU CERTO (3 bullets)
   - Destaques positivos, linguagem simples

3. O QUE PRECISA MELHORAR (2 bullets)
   - Pontos de atencao, sem alarmar

4. O QUE FAREMOS (3 bullets)
   - Proximas acoes concretas

5. NUMERO DO MES
   - 1 metrica em destaque grande (ex: "47 leads gerados")

Regras:
- MAXIMO 15 linhas
- Sem jargao tecnico (ou explique em parenteses)
- Tom confiante e profissional
- Pode ser enviado por WhatsApp
- Termine com: "Quer que eu detalhe algum ponto?"
```

**Exemplo de resultado esperado:** Resumo de 15 linhas com resultado principal, destaques, pontos de melhoria e proximas acoes.

### 8. Analise de ROI de Campanha

**Quando usar:** Para calcular e apresentar o retorno sobre investimento de campanhas para o cliente.

**Prompt:**
```
Preciso calcular e apresentar o ROI de uma campanha de forma clara para o cliente.

DADOS DA CAMPANHA:
- Cliente: [NOME DA EMPRESA]
- Campanha: [NOME DA CAMPANHA]
- Periodo: [PERIODO]
- Plataforma: [META ADS / GOOGLE ADS / AMBOS]

INVESTIMENTO:
- Valor investido em midia: R$ [VALOR]
- Seu fee de gestao: R$ [VALOR]
- Outros custos (producao, ferramentas): R$ [VALOR]
- Investimento total: R$ [VALOR]

RESULTADOS:
- Leads gerados: [NUMERO]
- Vendas fechadas: [NUMERO]
- Ticket medio: R$ [VALOR]
- Faturamento gerado: R$ [VALOR]
- Se nao tem dados de venda: [DESCREVA O QUE TEM — agendamentos, orcamentos, etc.]

Me entregue:

1. CALCULO DO ROI
   - Formula utilizada
   - ROI percentual
   - ROAS (retorno sobre gasto em midia)
   - Payback (em quanto tempo se paga)

2. ANALISE DO FUNIL COMPLETO
   - Custo por lead (CPL)
   - Taxa de conversao lead > venda
   - Custo por aquisicao (CPA)
   - Valor do cliente (LTV estimado, se possivel)

3. CONTEXTUALIZACAO
   - Esse ROI e bom para o nicho?
   - Benchmark do mercado para comparacao
   - Fatores que influenciaram positiva ou negativamente

4. CENARIOS
   - Se dobrar o investimento, projecao de resultado
   - Se otimizar a taxa de conversao em 20%, impacto no ROI
   - Ponto de equilibrio (breakeven)

5. APRESENTACAO VISUAL
   - Numeros formatados para impressionar mas sem exagerar
   - Grafico em texto do funil (topo > meio > fundo)
   - Destaque do numero mais impactante

Linguagem que o cliente entende: "Pra cada R$1 investido, voltaram R$X."
```

**Exemplo de resultado esperado:** Analise de ROI com calculos, benchmarks do mercado, cenarios de escala e apresentacao visual do funil.

### 9. Relatorio de Entregas do Mes

**Quando usar:** Para documentar tudo que foi entregue no mes e manter transparencia com o cliente.

**Prompt:**
```
Preciso montar um relatorio de entregas do mes para manter o cliente informado sobre tudo que foi produzido.

INFORMACOES:
- Cliente: [NOME DA EMPRESA]
- Periodo: [MES/ANO]
- Servico: [TRAFEGO / SOCIAL MEDIA / DESIGN / COMPLETO]

ENTREGAS DO MES:
[LISTE TUDO QUE FOI FEITO — exemplo:
- 20 posts para Instagram
- 8 stories
- 4 reels
- 10 copies de anuncio
- 2 campanhas novas
- 1 landing page
- 1 relatorio de performance
- 3 reunioes de alinhamento]

Monte um RELATORIO DE ENTREGAS com:

1. RESUMO QUANTITATIVO
   - Tabela: tipo de entrega | quantidade | status
   - Total de entregas no mes

2. DETALHAMENTO POR AREA
   Para cada area de trabalho:
   - O que foi entregue
   - Destaques (o que ficou especialmente bom)
   - Observacoes relevantes

3. COMPARATIVO COM O ESCOPO
   - O que foi combinado vs o que foi entregue
   - Entregas extras (acima do escopo)
   - Entregas pendentes (com justificativa e nova data)

4. HORAS/ESFORCO (opcional)
   - Estimativa de horas dedicadas por area
   - Demonstrar o valor do trabalho

5. PROXIMAS ENTREGAS
   - O que esta planejado para o proximo mes
   - Datas importantes
   - O que depende do cliente

Formate de forma que valorize o trabalho feito sem parecer que esta se gabando.
```

**Exemplo de resultado esperado:** Relatorio com tabela de entregas, detalhamento por area, comparativo com escopo e planejamento do proximo mes.

### 10. Previsao de Resultados Proximo Mes

**Quando usar:** Para projetar resultados e alinhar expectativas com o cliente antes do mes comecar.

**Prompt:**
```
Preciso criar uma projecao de resultados para o proximo mes baseada nos dados atuais.

DADOS HISTORICOS:
- Cliente: [NOME DA EMPRESA]
- Servico: [TIPO]
- Dados dos ultimos 3 meses:

MES 1 ([MES]):
[METRICAS]

MES 2 ([MES]):
[METRICAS]

MES 3 ([MES]):
[METRICAS]

VARIAVEIS DO PROXIMO MES:
- Orcamento planejado: R$ [VALOR] (mesmo / maior / menor que o atual)
- Mudancas estrategicas previstas: [DESCREVA]
- Sazonalidade: [ALTA / BAIXA / NORMAL para o periodo]
- Novos produtos/ofertas: [SIM/NAO]

Me entregue:

1. PROJECAO — CENARIO CONSERVADOR
   - Metricas projetadas (mantendo tendencia atual)
   - Premissas utilizadas

2. PROJECAO — CENARIO OTIMISTA
   - Metricas projetadas (com melhorias planejadas funcionando)
   - O que precisa acontecer para isso se concretizar

3. PROJECAO — CENARIO PESSIMISTA
   - Metricas projetadas (se algo der errado)
   - Riscos mapeados

4. META RECOMENDADA
   - A meta que eu recomendo (entre conservador e otimista)
   - Por que essa meta e realista
   - KPIs com numeros especificos

5. PLANO DE ACAO
   - O que faremos para atingir a meta
   - Testes planejados
   - Investimento sugerido
   - Timeline de implementacao

6. COMO VAMOS MEDIR
   - Checkpoints semanais
   - Quando pivotar se nao estiver no caminho

Formate como uma proposta de metas que o cliente possa aprovar.
```

**Exemplo de resultado esperado:** Projecao em 3 cenarios com metas numericas, plano de acao e checkpoints semanais.
