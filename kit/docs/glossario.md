# Glossário | Kit Piloto Automático com IA

## A

**Agente IA** — Um assistente virtual especializado que executa tarefas específicas. No kit, cada agente faz uma parte do trabalho (briefing, criação, revisão, entrega, relatório). Você conversa com ele e ele faz o trabalho.

**API** — Interface que conecta dois sistemas. Exemplo: a API do WhatsApp permite que o agente envie mensagens automaticamente. Você não precisa saber programar pra usar.

**Automação** — Qualquer processo que roda sozinho sem você precisar fazer na mão. Exemplo: o agente responde o WhatsApp às 7h da manhã enquanto você dorme.

## B

**Briefing** — Documento que resume tudo que precisa saber sobre um projeto. O agente de briefing monta isso automaticamente a partir de perguntas ao cliente.

## C

**Claude Code** — Ferramenta da Anthropic que permite usar agentes IA direto no seu computador. É o "motor" que faz os agentes do kit funcionarem. Gratuito pra instalar.

**CLI** — Interface de linha de comando. É o terminal/prompt de comando do seu computador. Onde você digita os comandos pra rodar o Claude Code.

**CPA** — Custo por aquisição. Quanto você gasta pra conseguir um cliente. CPA = investimento total / número de clientes.

**CPL** — Custo por lead. Quanto você gasta pra conseguir um contato. CPL = investimento / número de leads.

**CTA** — Call to Action. O botão ou frase que pede pro cliente fazer algo ("Compre agora", "Agende uma reunião").

## D

**Deploy** — Colocar algo no ar. "Fazer deploy da página" = publicar a página na internet.

## E

**Eugência** — Agência de uma pessoa só. Você faz tudo: prospecta, vende, entrega, atende, cobra. O kit existe pra resolver isso.

## F

**Follow-up** — Mensagem de acompanhamento. Mandar mensagem pro cliente depois de um contato inicial pra manter o relacionamento.

**Fluxo** — Sequência automática de ações. Exemplo: "fluxo de onboarding" = sequência de mensagens que o cliente novo recebe automaticamente.

## K

**KPI** — Key Performance Indicator. Métrica que mostra se algo está funcionando. Exemplo: taxa de abertura de email, ROAS, CPL.

## L

**Lead** — Contato de um potencial cliente. Alguém que demonstrou interesse.

**LP** — Landing page. Página de vendas onde o tráfego chega.

## M

**Markdown (.md)** — Formato de texto simples que os agentes usam. Abre em qualquer editor de texto. Os templates e prompts estão nesse formato.

## O

**Onboarding** — Processo de receber um cliente novo. Inclui: boas-vindas, coleta de informações, alinhamento de expectativas, primeiro entregável.

## P

**Placeholder** — Campo pra preencher. Nos templates aparece como `[NOME DO CLIENTE]` ou `[PREENCHER]`. Você substitui pelo dado real.

**Prompt** — Instrução que você dá pro agente IA. Quanto melhor o prompt, melhor o resultado. Os prompts do kit já estão prontos pra copiar e colar.

## R

**ROAS** — Return on Ad Spend. Quanto volta pra cada real investido em anúncio. ROAS 3 = cada R$1 investido gerou R$3 de receita.

**ROI** — Return on Investment. Retorno sobre investimento total (não só anúncio).

## S

**Screencast** — Gravação da tela do computador. Os vídeos do kit são screencasts mostrando como usar as ferramentas.

**Segmento** — Área de atuação. No kit: gestor de tráfego, social media, designer, videomaker.

**Squad** — Time/equipe. No contexto do kit, o "squad de agentes" é o seu time de agentes IA.

## T

**Template** — Modelo pronto pra preencher. Abre, coloca seus dados e entrega. Não precisa criar do zero.

**Terminal** — Programa do computador onde você digita comandos. No Windows: PowerShell ou CMD. No Mac: Terminal.

**Token** — Unidade de processamento da IA. Cada vez que você conversa com o agente, gasta tokens. O Claude Code tem um limite mensal.

## W

**Webhook** — Notificação automática entre sistemas. Exemplo: quando alguém compra, o webhook avisa o sistema de entrega pra liberar o acesso.
