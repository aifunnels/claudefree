# Prompts: Prospeccao e Qualificacao B2B

## Como usar
Copie o prompt, substitua as variaveis entre colchetes e cole na IA. Cada prompt resolve um problema especifico do processo de prospeccao.

---

## Prompt 1: Gerar Lista de Prospeccao por ICP

```
Atue como um SDR experiente em vendas B2B.

Minha empresa: [NOME_DA_SUA_EMPRESA]
O que vendemos: [PRODUTO/SERVICO]
ICP (cliente ideal): [DESCREVA O PERFIL -- segmento, tamanho, cargo do decisor, faturamento]
Regiao: [CIDADE/ESTADO/PAIS]

Com base nesse ICP, gere uma lista de 20 tipos de empresas que se
encaixam nesse perfil, incluindo:
- Segmento de atuacao
- Tamanho estimado (funcionarios e faturamento)
- Cargo do decisor que devo abordar
- Dor provavel que meu produto resolve
- Canal ideal de abordagem (email, LinkedIn, telefone, WhatsApp)

Organize em tabela. Priorize os segmentos com maior probabilidade
de conversao primeiro.
```

## Prompt 2: Criar Sequencia de Outbound (Cadencia Multicanal)

```
Atue como um especialista em outbound B2B que ja montou cadencias
para times de 50+ SDRs.

Minha empresa: [NOME_DA_SUA_EMPRESA]
Produto/servico: [O QUE VOCE VENDE]
ICP: [PERFIL DO CLIENTE IDEAL]
Proposta de valor principal: [RESULTADO QUE VOCE ENTREGA]
Case de sucesso: [NOME_EMPRESA_CASE] conseguiu [RESULTADO] em [PRAZO]

Crie uma cadencia de outbound de 14 dias com:
- 5 touchpoints minimo
- Alternando entre email, LinkedIn e telefone
- Cada mensagem com menos de 150 palavras
- Assunto de email que gere abertura (nao generico)
- CTA claro em cada touchpoint
- 1 break-up message no final

Para cada touchpoint, inclua:
- Dia (D+0, D+3, etc.)
- Canal
- Objetivo da mensagem
- Texto completo pronto para usar
- Variacao A/B para o primeiro email
```

## Prompt 3: Qualificar Lead com Perguntas BANT

```
Atue como um SDR que precisa qualificar um lead B2B usando o
framework BANT (Budget, Authority, Need, Timeline).

Contexto do lead:
- Empresa: [NOME_EMPRESA]
- Segmento: [SEGMENTO]
- Cargo do contato: [CARGO]
- Como chegou: [ORIGEM -- inbound, indicacao, evento, outbound]
- O que sabemos ate agora: [INFORMACOES DISPONIVEIS]

Gere:
1. 3 perguntas de BUDGET (orcamento) -- do mais sutil ao mais direto
2. 3 perguntas de AUTHORITY (decisor) -- identificar quem decide
3. 3 perguntas de NEED (necessidade) -- mapear dor real
4. 3 perguntas de TIMELINE (urgencia) -- entender o timing

Para cada pergunta, inclua:
- A pergunta em si
- Por que fazer essa pergunta (qual informacao ela extrai)
- Red flag na resposta (o que indica que NAO e um bom lead)
- Green flag na resposta (o que indica que E um bom lead)

No final, crie um scorecard simples de qualificacao:
- 0-3 pontos: Descartar
- 4-6 pontos: Nutrir
- 7-10 pontos: Agendar reuniao com closer
- 11-12 pontos: Prioridade maxima
```

## Prompt 4: Pesquisar Empresa Antes da Abordagem

```
Atue como um analista de inteligencia comercial B2B.

Preciso fazer uma pesquisa pre-abordagem sobre a empresa [NOME_EMPRESA].

Com base no que voce sabe (e no que eu forneco abaixo), monte um
briefing de abordagem com:

Informacoes que tenho:
- Site: [URL]
- Segmento: [SEGMENTO]
- Decisor: [NOME] -- [CARGO]
- LinkedIn do decisor: [URL_LINKEDIN]

Monte o briefing com:
1. RESUMO DA EMPRESA (2-3 frases)
   - O que fazem, pra quem, tamanho estimado

2. SINAIS DE OPORTUNIDADE
   - Contratacoes recentes (expansao?)
   - Noticias relevantes
   - Posts do decisor no LinkedIn (temas, dores, interesses)
   - Concorrentes diretos

3. DORES PROVAVEIS
   - 3 dores que empresas desse perfil costumam ter
   - Qual dor meu produto [PRODUTO] resolve melhor

4. GATILHOS DE CONTEXTO
   - 3 aberturas de mensagem personalizadas baseadas na pesquisa
   - Cada uma conectando algo especifico da empresa com minha solucao

5. OBJECOES ESPERADAS
   - 2 objecoes provaveis e como contornar
```

## Prompt 5: Criar Email de Reengajamento para Leads Frios

```
Atue como um copywriter especializado em vendas B2B.

Tenho uma lista de leads que:
- Foram abordados ha [TEMPO] atras
- Nao responderam OU responderam "nao e o momento"
- Sao do segmento [SEGMENTO]
- O ICP continua valido

Minha empresa: [NOME_DA_SUA_EMPRESA]
O que vendemos: [PRODUTO/SERVICO]
Novidade recente: [ALGO NOVO -- case, feature, conteudo, evento, resultado]

Crie 3 emails de reengajamento, cada um com abordagem diferente:

EMAIL 1: Abordagem por novidade
- "Desde a ultima vez que conversamos, [NOVIDADE]"
- Foco em mostrar que algo mudou

EMAIL 2: Abordagem por conteudo de valor
- Oferecer um recurso gratuito (checklist, benchmark, estudo)
- Nao pedir nada em troca

EMAIL 3: Abordagem por case de resultado
- Mostrar resultado concreto de cliente similar
- Gerar curiosidade sobre como foi feito

Para cada email:
- Assunto (2 variacoes)
- Corpo (menos de 120 palavras)
- CTA (1 acao clara)
- Melhor dia/horario para enviar
```
