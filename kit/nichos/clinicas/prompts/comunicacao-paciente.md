# Prompts: Comunicacao com Paciente

> 5 prompts para melhorar a comunicacao da clinica com seus pacientes em diferentes momentos do relacionamento.
> Copie, cole e adapte com os dados da sua clinica.

---

## Prompt 1 -- Mensagens de confirmacao e lembrete personalizadas

```
Crie um conjunto de mensagens de WhatsApp para confirmar e lembrar
pacientes sobre suas consultas. Preciso de variacoes para nao enviar
sempre o mesmo texto.

Dados:
- Nome da clinica: [Nome]
- Horario de funcionamento: [ex: seg-sex 8h-18h]

Crie 3 variacoes para cada momento:

1. CONFIRMACAO (enviada ao agendar):
   - Tom: caloroso, organizado
   - Incluir: data, horario, endereco, profissional, o que trazer

2. LEMBRETE 48H ANTES:
   - Tom: leve, solicito
   - Incluir: data, horario, opcao de confirmar ou remarcar

3. LEMBRETE 2H ANTES:
   - Tom: breve, animado
   - Incluir: horario, endereco resumido

4. MENSAGEM DE NO-SHOW (paciente faltou):
   - Tom: compreensivo, sem culpa
   - Oferecer remarcacao facil

Regras:
- Mensagens curtas (maximo 3 paragrafos)
- Sempre incluir opcao de remarcar
- Nunca parecer cobranca ou pressao
- Personalizar com nome do paciente
```

## Prompt 2 -- Respostas para duvidas frequentes

```
Sou recepcionista de uma clinica de [especialidade] e preciso de
respostas prontas para as duvidas mais comuns dos pacientes via
WhatsApp.

Crie respostas profissionais e acolhedoras para:

1. "Quanto custa a consulta/procedimento?"
2. "Voces aceitam [nome do convenio]?"
3. "Quanto tempo dura o procedimento?"
4. "Doi? Tem anestesia?"
5. "Qual o tempo de recuperacao?"
6. "Posso parcelar?"
7. "Preciso de encaminhamento medico?"
8. "Tem estacionamento?"
9. "Posso levar acompanhante?"
10. "Como funciona o agendamento?"

Para cada resposta:
- Ser objetiva (3-4 linhas no maximo)
- Tom acolhedor, nao robotico
- Quando a pergunta exigir avaliacao presencial, orientar gentilmente
  que cada caso e unico e que a melhor forma e agendar uma avaliacao
- Nunca dar diagnostico ou prometer resultado especifico
- Finalizar com proximo passo (ex: "Posso agendar sua avaliacao?")
```

## Prompt 3 -- Comunicacao de mudancas e novidades

```
Preciso comunicar aos pacientes da clinica [Nome] sobre:

[ESCOLHER O CENARIO:]

A) Novo profissional na equipe
B) Mudanca de endereco
C) Novo procedimento/equipamento disponivel
D) Mudanca de horario de funcionamento
E) Ferias/recesso da clinica

Dados especificos:
[Preencher conforme o cenario escolhido]

Crie:
1. Mensagem para WhatsApp (lista de transmissao) -- ate 500 caracteres
2. Post para Instagram (texto + sugestao de visual)
3. E-mail informativo (para base de pacientes cadastrados)

Tom: transparente, profissional, positivo. O paciente deve sentir
que a mudanca e boa ou que esta sendo bem informado.

Incluir:
- O que muda
- A partir de quando
- Como afeta o paciente
- Canal para duvidas
```

## Prompt 4 -- Pos-consulta: orientacoes por procedimento

```
Crie mensagens de orientacao pos-procedimento para enviar ao paciente
via WhatsApp apos a consulta.

Procedimento realizado: [ESCOLHER UM:]

A) Limpeza dental
B) Extracao de dente
C) Preenchimento labial / harmonizacao
D) Peeling quimico
E) Consulta medica com prescricao
F) Procedimento cirurgico ambulatorial
G) [Outro -- especificar]

A mensagem deve conter:
1. Saudacao pos-procedimento (empatica)
2. Orientacoes especificas de cuidado (em lista, facil de ler)
3. Sinais de alerta (quando procurar atendimento)
4. Contato de emergencia da clinica
5. Lembrete do proximo retorno (se aplicavel)

Regras:
- Linguagem simples e direta
- Nao substituir a orientacao presencial do profissional
- Servir como lembrete do que foi explicado na consulta
- Incluir: "Em caso de duvida, entre em contato: [telefone]"
```

## Prompt 5 -- Pesquisa de satisfacao conversacional

```
Crie uma pesquisa de satisfacao por WhatsApp para a clinica [Nome],
no formato de conversa (nao formulario).

A pesquisa deve parecer uma conversa natural, nao um questionario frio.

Estrutura:
1. Abertura calorosa (1 mensagem)
2. Pergunta sobre o atendimento da recepcao (escala 1-5 ou emoji)
3. Pergunta sobre o profissional que atendeu (escala 1-5 ou emoji)
4. Pergunta sobre a estrutura da clinica (escala 1-5 ou emoji)
5. Pergunta aberta: "O que podemos melhorar?"
6. Pergunta: "Voce indicaria nossa clinica para alguem proximo?"
   (NPS -- escala 0 a 10)
7. Agradecimento genuino

Regras:
- Maximo 7 mensagens no total (nao cansar o paciente)
- Cada mensagem so e enviada apos a resposta anterior
- Se a nota for baixa (1-2), encaminhar para atendimento humano
- Usar emojis com moderacao
- Incluir opcao de pular a pesquisa

Incluir tambem:
- Mensagem para pacientes que deram NPS 9-10 pedindo avaliacao no Google
- Mensagem para pacientes que deram NPS 0-6 oferecendo conversa com gestor
```
