# Prompts: Captacao de Pacientes

> 5 prompts prontos para atrair novos pacientes para clinicas medicas, odontologicas e esteticas.
> Copie, cole e adapte com os dados da sua clinica.

---

## Prompt 1 -- Criar anuncios para Instagram/Facebook

```
Voce e um copywriter especializado em marketing para clinicas de saude.

Crie 5 variacoes de anuncio para Instagram/Facebook para a seguinte clinica:

- Nome da clinica: [Nome]
- Especialidade principal: [ex: odontologia, dermatologia, estetica]
- Procedimento em destaque: [ex: limpeza dental, harmonizacao facial, check-up]
- Publico-alvo: [ex: mulheres 25-45 anos, empresarios, maes]
- Diferencial da clinica: [ex: atendimento humanizado, tecnologia de ponta, localizacao]
- Cidade/regiao: [ex: Zona Sul de SP]

Para cada variacao, inclua:
- Headline (ate 40 caracteres)
- Texto principal (ate 125 palavras)
- CTA (chamada para acao)

Regras:
- Tom acolhedor e profissional, nunca agressivo
- Nao fazer promessas de resultado
- Nao mencionar precos nos anuncios (direcionar para contato)
- Focar na experiencia do paciente e no beneficio emocional
- Respeitar normas do CFM/CFO para publicidade medica/odontologica
```

## Prompt 2 -- Script de primeiro atendimento no WhatsApp

```
Crie um script de atendimento para a recepcionista usar no WhatsApp
quando um novo paciente entra em contato pela primeira vez.

Contexto:
- Clinica: [Nome]
- Especialidades: [listar]
- Horarios de funcionamento: [ex: seg-sex 8h-18h, sab 8h-12h]
- Aceita convenio: [sim/nao -- quais]
- Estacionamento: [sim/nao]

O script deve cobrir:
1. Saudacao calorosa
2. Perguntar o motivo do contato
3. Coletar nome e telefone
4. Informar sobre agendamento (disponibilidade)
5. Responder duvidas frequentes (localizacao, convenio, estacionamento)
6. Encerramento com proximo passo claro

Tom: acolhedor, profissional, eficiente. Como se fosse uma conversa
natural, nao um robo.

Inclua tambem 5 respostas rapidas para objecoes comuns:
- "Quanto custa?"
- "Aceita meu convenio?"
- "Tem horario a noite?"
- "E longe de [bairro]?"
- "Preciso de encaminhamento?"
```

## Prompt 3 -- Criar conteudo educativo para redes sociais

```
Crie um calendario de conteudo para 2 semanas (10 posts) para o
Instagram de uma clinica de [especialidade].

Para cada post, inclua:
- Formato sugerido (carrossel, reels, stories, post unico)
- Titulo/hook
- Roteiro resumido ou texto do post
- Hashtags relevantes (5-8)

Categorias de conteudo a alternar:
- Educativo (explicar procedimento ou cuidado de saude)
- Bastidores (dia a dia da clinica, equipe)
- Depoimento/caso (sem identificar paciente, respeitar etica)
- Dica pratica (algo que o seguidor pode aplicar hoje)
- Humanizacao (historia do profissional, missao da clinica)

Regras:
- Linguagem acessivel (nao usar jargao medico sem explicar)
- Nao fazer diagnostico ou recomendar tratamento via post
- Incluir CTA suave no final (ex: "Ficou com duvida? Mande mensagem")
- Respeitar normas de publicidade do CFM/CFO
- Tom: educativo, acolhedor, confiavel
```

## Prompt 4 -- Programa de indicacao entre pacientes

```
Crie um programa de indicacao para a clinica [Nome] que incentive
pacientes satisfeitos a indicar amigos e familiares.

Inclua:
1. Nome criativo para o programa
2. Mecanica (como funciona a indicacao)
3. Beneficio para quem indica (ex: desconto, brinde, prioridade)
4. Beneficio para quem e indicado (ex: avaliacao gratuita, desconto)
5. Regras claras (limite de indicacoes, validade, como resgatar)
6. Mensagem modelo para o paciente enviar ao indicado
7. Mensagem de agradecimento quando a indicacao converte

Restricoes:
- O programa nao pode condicionar beneficio a realizacao de
  procedimento (etica medica)
- Nao usar linguagem de "venda" ou "comissao"
- Focar em compartilhar uma boa experiencia, nao em ganhar algo
- Manter conformidade com LGPD (consentimento do indicado)
```

## Prompt 5 -- Otimizar o perfil do Google Meu Negocio

```
Sou gestor de uma clinica de [especialidade] em [cidade/bairro].

Me ajude a otimizar o perfil do Google Meu Negocio para atrair
mais pacientes locais.

Dados da clinica:
- Nome: [Nome]
- Especialidades: [listar]
- Diferenciais: [listar 3]
- Publico principal: [descrever]

Preciso de:
1. Descricao otimizada para o perfil (ate 750 caracteres)
   - Incluir palavras-chave relevantes naturalmente
   - Focar em beneficios para o paciente
   - Incluir CTA

2. Lista de 10 categorias relevantes para selecionar no Google

3. 5 perguntas frequentes (FAQ) para adicionar no perfil
   com respostas prontas

4. Estrategia para conseguir mais avaliacoes:
   - Melhor momento para pedir avaliacao
   - Mensagem modelo para solicitar
   - Como responder avaliacoes positivas e negativas

5. 5 ideias de posts para o Google Meu Negocio
   (o Google permite posts semanais no perfil)

Tom: profissional e acolhedor. Otimizar para SEO local sem
parecer robotico.
```
