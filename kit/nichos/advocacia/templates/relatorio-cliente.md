# Template: Relatorio de Andamento para o Cliente

## Instrucao de Uso

Envie este relatorio periodicamente ao cliente para mante-lo informado sobre o andamento do caso. A frequencia recomendada depende da fase processual: mensal para processos em andamento regular, semanal para fases criticas (audiencias, prazos de recurso).

Preencha os campos entre colchetes. Adapte o tom conforme o perfil do cliente: mais tecnico para clientes empresariais, mais acessivel para pessoas fisicas.
