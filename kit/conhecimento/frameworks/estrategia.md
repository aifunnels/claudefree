# Strategy Framework — CEO Squad

## SWOT e Cruzamentos

### O que e SWOT
Ferramenta de diagnostico estrategico que mapeia o contexto interno (forcas e fraquezas) e externo (oportunidades e ameacas) do negocio.

### Construcao do SWOT

**Forcas (Strengths)** — internas, positivas
- O que o negocio faz melhor que os concorrentes
- Recursos unicos (pessoas, tecnologia, processo, marca)
- Vantagens competitivas sustentaveis

**Fraquezas (Weaknesses)** — internas, negativas
- Onde o negocio esta abaixo do benchmark
- Gargalos operacionais ou comerciais
- Lacunas de capacidade ou recurso

**Oportunidades (Opportunities)** — externas, positivas
- Tendencias de mercado favoraveis
- Necessidades do cliente nao atendidas
- Movimentos de concorrentes que abrem espaco

**Ameacas (Threats)** — externas, negativas
- Mudancas regulatorias ou macroeconomicas
- Novos entrantes ou substitutos
- Risco de concentracao em cliente, canal ou produto

### Cruzamentos SWOT (matriz de estrategias)

| Tipo | Logica | Pergunta-guia |
|------|--------|--------------|
| **SO** (Forca + Oportunidade) | Usar forca para capturar oportunidade | "Com nossa forca X, como capturamos a oportunidade Y?" |
| **ST** (Forca + Ameaca) | Usar forca para mitigar ameaca | "Com nossa forca X, como neutralizamos a ameaca Y?" |
| **WO** (Fraqueza + Oportunidade) | Superar fraqueza para capturar oportunidade | "Se resolvermos a fraqueza X, conseguimos capturar Y?" |
| **WT** (Fraqueza + Ameaca) | Minimizar fraqueza para nao ser atacado | "Como evitamos que nossa fraqueza X seja explorada pela ameaca Y?" |

## Porter's 5 Forces

Framework para analisar a atratividade e competitividade do setor.

| Forca | O que analisa | Impacto no negocio |
|-------|-------------|-------------------|
| **Rivalidade entre concorrentes** | Intensidade da competicao atual | Alta rivalidade comprime margens |
| **Ameaca de novos entrantes** | Facilidade de entrar no setor | Baixa barreira = mais pressao futura |
| **Poder de barganha dos fornecedores** | Dependencia de poucos fornecedores | Alta dependencia = vulnerabilidade de custo |
| **Poder de barganha dos clientes** | Concentracao de clientes, opcoes alternativas | Alto poder = pressao por desconto |
| **Ameaca de produtos substitutos** | Alternativas que resolvem o mesmo problema | Forte substituto = teto de preco |

### Interpretacao

| Nivel | Significado |
|-------|-------------|
| 5 forcas fracas | Setor altamente atrativo — margens sustentaveis |
| Forcas mistas | Setor moderado — estrategia de nicho ou diferenciacão |
| 5 forcas fortes | Setor pouco atrativo — requer vantagem competitiva forte |

## Metodologia OKR

### Estrutura

```
Objective (O)
├── Key Result 1 (KR1)
├── Key Result 2 (KR2)
└── Key Result 3 (KR3)
```

**Objective:** Qualitativo, inspirador, descreve o estado desejado ao final do periodo.
**Key Result:** Quantitativo, mensuravel, indica se o Objective foi atingido.

### Regras de Ouro

| Regra | Descricao |
|-------|-----------|
| Maximo 3 OKRs por ciclo | Foco > completude |
| KR com baseline e meta | "De X para Y ate data Z" |
| KR binario ou numerico | Sim/Nao ou numero — nunca subjetivo |
| Responsavel por OKR | Um dono, nao um comite |
| Cadencia de revisao | Semanal ou quinzenal |
| Transparencia total | Todos veem todos os OKRs |

### Exemplos de KR Correto vs Errado

| Errado | Correto |
|--------|---------|
| "Melhorar vendas" | "Aumentar MRR de R$50k para R$75k" |
| "Otimizar processos" | "Reduzir tempo de entrega de 15 para 7 dias" |
| "Ser mais conhecido" | "Atingir 10.000 seguidores organicos no LinkedIn" |
| "Melhorar retencao" | "Aumentar taxa de renovacao de 60% para 80%" |

## Business Model Canvas

Framework de 9 blocos para mapear e analisar o modelo de negocio.

| Bloco | Pergunta |
|-------|----------|
| **Segmentos de Clientes** | Para quem criamos valor? |
| **Proposta de Valor** | Que problema resolvemos? Como somos diferentes? |
| **Canais** | Como chegamos ate os clientes? |
| **Relacionamento com Clientes** | Como nos relacionamos? |
| **Fontes de Receita** | Como monetizamos? |
| **Recursos-Chave** | O que precisamos para operar? |
| **Atividades-Chave** | O que fazemos de essencial? |
| **Parceiros-Chave** | De quem dependemos? |
| **Estrutura de Custos** | Quais sao os principais custos? |

### Pontos de Analise

- Coerencia: a proposta de valor e sustentada pelos recursos e atividades?
- Concentracao: ha dependencia de um unico segmento, canal ou fonte de receita?
- Escalabilidade: o modelo permite crescer sem aumentar custos proporcionalmente?
- Diferenciacão: a proposta de valor e replicavel pelos concorrentes?

## Matriz Impacto x Esforco

Framework de priorizacao para iniciativas e decisoes.

```
         Impacto
    Alta |  Quick  |  Big  |
         |  Wins   |  Bets |
         |---------|-------|
    Baixa|  Fill   | Thank |
         |  Ins    |  Yous |
         |---------|-------|
          Esforco   Esforco
           Baixo     Alto
```

| Quadrante | Acao |
|-----------|------|
| Alta impacto + Baixo esforco (Quick Wins) | Prioridade maxima — executar agora |
| Alta impacto + Alto esforco (Big Bets) | Planejar e alocar recursos — executar com cuidado |
| Baixo impacto + Baixo esforco (Fill-ins) | Executar quando houver capacidade ociosa |
| Baixo impacto + Alto esforco (Thank Yous) | Evitar — custo alto, retorno baixo |

### Calculo de Score
`Score = Impacto (1-5) / Esforco (1-5)`
- Score > 2.0: Quick Win (priorizar)
- Score 1.0-2.0: avaliar contexto
- Score < 1.0: entrar no backlog ou descartar

## Visao e Posicionamento

### Elementos de uma Visao Estrategica

| Elemento | O que define |
|----------|-------------|
| Horizonte temporal | 12 meses (operacional), 3 anos (estrategico), 10 anos (aspiracional) |
| Estado desejado | Onde estaremos em metricas concretas |
| Apostas estrategicas | As 2-3 prioridades que mudam o jogo |
| Stop-doing list | O que vamos parar de fazer para focar |

### Posicionamento Competitivo

Framework STP (Segmentacao, Targeting, Posicionamento):

**Segmentacao:** Quem sao os diferentes grupos de clientes no mercado?
**Targeting:** Para qual segmento temos mais vantagem competitiva?
**Posicionamento:** Como queremos ser percebidos por esse segmento vs concorrentes?

Template de posicionamento:
```
Para [segmento-alvo] que [necessidade ou problema],
[nome do negocio] e um [categoria do produto/servico]
que [proposta de valor principal].
Diferente de [alternativa principal],
[nome do negocio] [diferenciador chave].
```
