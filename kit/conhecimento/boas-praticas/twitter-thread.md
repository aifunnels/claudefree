## Platform Rules

- Threads receive 2-3x more impressions than standalone tweets.
- Early engagement on the first tweet is critical.
- Hashtags only on first tweet.

## Content Structure

1. **Hook tweet** — Standalone compelling statement.
2. **Context tweet** — Why this matters.
3. **Body tweets** — One insight per tweet, numbered.
4. **Summary tweet** — Key takeaways in bullets.
5. **CTA tweet** — Retweet tweet 1, follow, bookmark.

## Writing Guidelines

- First tweet must work as standalone.
- Number every tweet.
- One point per tweet.
- 5-15 tweets total.

## Quality Criteria

- [ ] First tweet works standalone
- [ ] Thread is 5-15 tweets
- [ ] Every tweet has one clear point
- [ ] Tweets are numbered
- [ ] Hashtags only on first tweet
- [ ] Summary tweet with bullet recap
- [ ] Final tweet has CTA
