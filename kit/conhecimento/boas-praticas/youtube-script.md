## Platform Rules

- CTR + Watch Time are the two most important metrics.
- First 30 seconds determine if viewers stay.
- Thumbnail is 50% of performance.
- 8-15 minute videos are optimal.
- Best upload: Thu/Fri/Sat 2-4 PM.

## Content Structure

1. **Hook (0-30s)** — State premise, preview payoff, create curiosity loop.
2. **Intro (30s-1min)** — Brief context.
3. **Body (1min-Xmin)** — 3-5 sections, pattern interrupts every 30-60s.
4. **Climax (60-70% through)** — Most valuable insight.
5. **Summary (last 1-2min)** — Recap key points.
6. **CTA + End Screen (last 20s)** — One primary CTA.

## Writing Guidelines

- Thumbnail first, before scripting.
- Title: curiosity gap + specific benefit.
- Hook establishes stakes in 30 seconds.
- Write for spoken delivery.
- Add timestamps in description.

## Quality Criteria

- [ ] Thumbnail concept defined before script
- [ ] Title 50-70 characters with curiosity + benefit
- [ ] Hook creates open loop within 30 seconds
- [ ] Pattern interrupt cues every 30-60 seconds
- [ ] 3-5 body sections with subheadings
- [ ] Climax at 60-70% through
- [ ] Duration 8-15 minutes
- [ ] Description includes timestamps
- [ ] End screen with one primary CTA
