## Core Principles

1. **Insight over raw data.** Every metric must have a business implication.
2. **Always contextualize.** Compare against previous period, benchmark, or target.
3. **Confidence levels on every finding.** High / Medium / Low.
4. **Structured output format.** Executive Summary, Metrics, Insights, Recommendations, Methodology.
5. **Cross-reference data sources.** Flag divergences >10%.
6. **Metric priority weighting.** Actionable metrics over vanity metrics.
7. **Escalation for anomalies.** Flag >25% movement as critical.
8. **Methodology transparency.** State time period, sources, sample sizes, exclusions.

## Analysis Steps

1. Data Collection
2. Pattern Identification (>15% movement)
3. Benchmarking (historical, industry, internal)
4. Insight Synthesis ("What happened", "Why it matters", "What it suggests")
5. Recommendation Formation (action, impact, confidence, effort, priority)
6. Report Compilation

## Quality Criteria

- [ ] Every metric has at least one comparison column
- [ ] Every insight includes business implication
- [ ] Every recommendation has 5 required fields
- [ ] Executive summary has exactly 3 bullet points
- [ ] Confidence levels assigned to all insights and recommendations
- [ ] Anomalies >25% flagged as critical
- [ ] Methodology section present
