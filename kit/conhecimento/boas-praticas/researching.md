## Core Principles

1. **Source verification first.** Never include a finding without verifying against at least one additional source.
2. **Freshness bias.** Prefer recent sources.
3. **Primary over secondary.** Original reports over blog posts.
4. **Structured output.** Key Findings, Trending Angles, Sources, Recommendations, Gaps.
5. **Contradiction surfacing.** Present both sides when sources disagree.
6. **Access date logging.** Record when each source was accessed.

## Methodology

1. Information Landscape Mapping
2. Focused Search Sweep (5-10 candidates)
3. Deep-Dive (3-5 best sources)
4. Cross-Referencing (High/Medium/Low confidence)
5. Synthesis into Brief Format
6. Self-Review Checklist

## Quality Criteria

- [ ] All findings include source URLs and access dates
- [ ] Confidence levels assigned to every finding
- [ ] 2+ sources corroborate each high-confidence finding
- [ ] Trending angles include lifecycle assessment
- [ ] Sources table with type and relevance score
- [ ] Gaps section populated
- [ ] Recommendations are actionable
- [ ] No opinions presented as facts
- [ ] Contradictory evidence surfaced
