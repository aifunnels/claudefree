# KPI Reference — Benchmarks Comerciais por Segmento

## Benchmarks de Conversao por Estagio

### Funil Geral B2B (SaaS / Servico)

| Estagio | Taxa Benchmark | Variacao por Ticket |
|---------|---------------|---------------------|
| Prospecto → Qualificado (CDP) | 15-25% | Menor para ticket alto |
| Qualificado → Discovery realizada | 40-60% | |
| Discovery → Proposta enviada | 50-70% | |
| Proposta → Negociacao | 40-60% | |
| Negociacao → Fechado Ganho | 50-70% | |
| **Win rate global** | **15-30%** | Sobre total de leads |

| Ticket | Win Rate Esperado | Ciclo Medio | Deals/Closer/mes |

## Metricas de SDR (Prospecao)

### Benchmarks por Canal

| Canal | Taxa de Resposta | SQL Rate | Notas |
|-------|----------------|---------|-------|
| LinkedIn (mensagem direta) | 20-40% | 5-15% | Maior taxa de resposta |
| Email frio | 3-15% | 1-5% | Varia muito por personalizacao |
| Cold call | 5-10% de conexao | 2-8% de SQL | Mais efetivo para enterprise |
| Indicacao | 50-80% de resposta | 20-40% de SQL | Melhor ROI |
