## Benchmarks de Automacao

| Metrica | Referencia |
|---------|-----------|
| Payback aceitavel | <6 meses |
| ROI minimo para justificar build | 3x investimento em 12 meses |
| % processos repetitivos automatizados (meta) | >70% |
| Tempo medio de build (automacao simples) | 4-8h |
| Tempo medio de build (automacao complexa) | 16-40h |

## Benchmarks de Capacidade

| Situacao | Acao Recomendada |
|---------|-----------------|
| Utilizacao >90% por 2+ semanas | Contratar ou redistribuir urgente |
| Utilizacao >85% por 1 mes | Planejar contratacao ou automacao |
| Utilizacao 70-85% | Monitorar — saudavel mas sem folga |
| Utilizacao 40-70% | Ideal — espaco para crescimento |
| Utilizacao <40% | Investigar — subutilizacao ou processo mal medido |
