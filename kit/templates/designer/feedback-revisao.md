# Template de Feedback e Revisao

> **Como usar:** Envie ao cliente junto com a entrega das pecas. O cliente preenche com feedback estruturado, indicando o que aprovar, o que ajustar e a prioridade. Elimina feedbacks vagos como "nao gostei" e agiliza o processo de revisao.

---

## Instrucoes para o Cliente

Preencha a tabela abaixo para cada peca entregue:

1. **Status:** Escolha entre APROVADO, AJUSTAR ou REFAZER
2. **O que ajustar:** Seja especifico (ex: "trocar cor do fundo para azul mais escuro", nao "nao gostei da cor")
3. **Prioridade:** Alta = precisa mudar antes de publicar / Media = seria bom mudar / Baixa = opcional
4. **Referencia:** Se tiver, cole o link de como voce quer que fique

## Guia de Feedback Eficiente

Para agilizar as revisoes, siga estas dicas:

| Em vez de... | Diga... |
|-------------|---------|
| "Nao gostei" | "A cor de fundo esta muito escura, prefiro um tom mais claro como #F5F0E8" |
| "Muda a fonte" | "Troque a fonte do titulo para algo mais moderno, estilo sans-serif como Montserrat" |
| "Nao era isso" | "O layout deveria ter a foto a esquerda e o texto a direita, como nesta referencia: [link]" |
| "Pode melhorar" | "A imagem do produto esta pequena. Aumentar para ocupar 60% do espaco" |
| "Faz de novo" | "O conceito nao transmite premium. Referencia de estilo que busco: [link]. Cores mais escuras e fonte serifada." |
