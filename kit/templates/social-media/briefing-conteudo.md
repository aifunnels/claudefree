# Briefing de Conteudo

> **Como usar:** Preencha para cada peca de conteudo que precisar ser produzida. Envie ao designer, videomaker ou copywriter com todas as informacoes necessarias para evitar retrabalho.

---

## Informacoes Basicas

| Campo | Valor |
|-------|-------|
| **Cliente** | [NOME DO CLIENTE] |
| **Data de entrega** | [DD/MM/AAAA] |
| **Data de publicacao** | [DD/MM/AAAA] |
| **Plataforma** | [Instagram / TikTok / LinkedIn / YouTube / Facebook] |
| **Formato** | [Feed / Reels / Stories / Carrossel / Video / Live] |
| **Responsavel pela producao** | [NOME DO DESIGNER/VIDEOMAKER] |
| **Aprovador** | [NOME DE QUEM APROVA] |

---

## Sobre o Conteudo

| Campo | Valor |
|-------|-------|
| **Tema** | [ASSUNTO PRINCIPAL DO POST] |
| **Pilar de conteudo** | [Educativo / Entretenimento / Autoridade / Venda / Bastidores / Comunidade] |
| **Objetivo** | [Engajamento / Alcance / Conversao / Educacao / Prova social] |
| **Publico-alvo** | [QUEM DEVE VER ESSE CONTEUDO — perfil, dor, momento] |

## Checklist de Aprovacao

- [ ] Copy revisada e aprovada
- [ ] Visual alinhado com identidade do cliente
- [ ] Dimensoes corretas para a plataforma
- [ ] CTA claro e visivel
- [ ] Logo/marca d'agua presente
- [ ] Sem erros de portugues
- [ ] Arquivo final nos formatos corretos (PNG, MP4, etc.)
