# Checklist de Producao de Video

> **Como usar:** Preencha antes de cada gravacao. Cobre equipamento, locacao, roteiro, figurino, backup e pos-producao. Leve impresso ou no celular no dia da gravacao para nao esquecer nada.

---

| Campo | Valor |

---

## Pre-Producao (antes do dia da gravacao)

### Planejamento

- [ ] Roteiro finalizado e aprovado
- [ ] Storyboard pronto (se aplicavel)
- [ ] Horarios definidos e confirmados com todos os envolvidos
- [ ] Locacao visitada previamente (ou fotos recebidas)
- [ ] Iluminacao natural verificada (horario do sol na locacao)
- [ ] Autorizacao de uso da locacao (se externa)
- [ ] Autorizacao de imagem assinada (se aparecer terceiros)

### Comunicacao com o Cliente

- [ ] Cliente sabe data, horario e local
- [ ] Cliente sabe o que vestir (figurino alinhado)
- [ ] Cliente recebeu o roteiro/pauta para se preparar
- [ ] Acessos confirmados (estacionamento, portaria, chave)

---

## Equipamento (verificar no dia anterior)

### Camera

- [ ] Camera principal carregada e funcionando
- [ ] Camera secundaria (se usar multi-angulo)
- [ ] Lentes necessarias: [LISTA — ex: 24-70mm, 50mm]
- [ ] Cartao de memoria formatado e com espaco suficiente
- [ ] Cartao de memoria reserva
- [ ] Bateria principal 100%
- [ ] Bateria reserva 100%
- [ ] Tripe / Estabilizador / Gimbal

### Audio

- [ ] Microfone lapela (bateria carregada)
- [ ] Microfone shotgun / boom (se usar)
- [ ] Gravador externo (se usar — ex: Zoom H1n)
- [ ] Cabo reserva
- [ ] Espuma antivento (se externo)
- [ ] Teste de audio feito (sem ruido, volume OK)

### Iluminacao

- [ ] Ring light / Softbox / LED panel
- [ ] Extensao eletrica (verificar tomada na locacao)
- [ ] Rebatedor (se usar luz natural)
- [ ] Filtro ND (se externo com sol forte)

### Extras

- [ ] Fita gaffer / fita crepe
- [ ] Prendedores (prender lapela, ajustar figurino)
- [ ] Lenco de limpeza para lente
- [ ] Power bank (para celular)
- [ ] Teleprompter / tablet com roteiro (se necessario)

---

## No Local (antes de comecar a gravar)

### Setup

- [ ] Cenario montado e enquadrado
- [ ] Fundo limpo (sem objetos distrativos)
- [ ] Iluminacao posicionada e testada
- [ ] Sombras indesejadas eliminadas
- [ ] Camera nivelada e enquadrada
- [ ] Foco testado (foco automatico ou manual travado)
- [ ] White balance ajustado
- [ ] Resolucao e FPS configurados: [RESOLUCAO] / [FPS]

### Audio

- [ ] Teste de audio gravado (10 segundos de fala + 10 segundos de silencio)
- [ ] Ruidos ambientes minimizados (ar condicionado, rua, obras)
- [ ] Microfone posicionado corretamente (lapela a 20cm da boca)
- [ ] Monitorando audio com fone durante a gravacao

### Gravacao

- [ ] Claquete ou referencia de audio no inicio (para sincronizar)
- [ ] Gravar 5 segundos antes e depois de cada take
- [ ] Verificar a cada 3-4 takes se o material esta bom (playback rapido)
- [ ] Gravar takes extras de seguranca para cenas importantes

---

## Durante a Gravacao

### Para cada video/cena

- [ ] Cena [1] gravada — takes: [NUMERO]
- [ ] Cena [2] gravada — takes: [NUMERO]
- [ ] Cena [3] gravada — takes: [NUMERO]
- [ ] Cena [4] gravada — takes: [NUMERO]
- [ ] Cena [5] gravada — takes: [NUMERO]

### B-roll (imagens de cobertura)

- [ ] B-roll ambiente / locacao
- [ ] B-roll produto / servico
- [ ] B-roll detalhes (maos, objetos, tela)
- [ ] B-roll de bastidores (para stories/conteudo extra)

---

## Pos-Gravacao (no mesmo dia)

### Backup

- [ ] Arquivos copiados para HD externo
- [ ] Arquivos copiados para nuvem (Google Drive / Dropbox)
- [ ] Verificar se todos os arquivos abrem corretamente
- [ ] Cartao de memoria NAO formatado ate confirmar backup

### Organizacao de Arquivos

```
[CLIENTE]_[PROJETO]_[DATA]/
├── camera-principal/
│   ├── cena-01/
│   ├── cena-02/
│   └── b-roll/
├── camera-secundaria/ (se tiver)
├── audio-externo/ (se tiver)
├── fotos-bastidores/
└── notas-gravacao.txt
```

- [ ] Pastas criadas e arquivos organizados
- [ ] Melhores takes marcados (favoritar ou renomear)
- [ ] Notas de gravacao salvas (o que deu certo, o que ajustar)

---

## Pos-Producao

- [ ] Selecao dos melhores takes
- [ ] Sincronizacao de audio (se gravador externo)
- [ ] Edicao conforme storyboard/roteiro
- [ ] Color grading aplicado
- [ ] Legendas adicionadas
- [ ] Musica e efeitos sonoros
- [ ] Revisao final em tela cheia
- [ ] Exportacao nos formatos corretos
- [ ] Entrega ao cliente / social media manager
