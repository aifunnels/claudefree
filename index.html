04 — Install

Setup em 1 linha

Uma linha no terminal. Instala tudo: OpenClaude, Ollama, modelo, ripgrep, Kit PA com 14 agentes. Cria o projeto na Area de Trabalho.

powershellone-liner
irm https://raw.githubusercontent.com/aifunnels/claudefree/master/scripts/setup-windows.ps1 | iex
Note
Abra o PowerShell (nao Git Bash), cole a linha acima e pressione Enter. O script detecta o que ja esta instalado e instala o que falta.
bashone-liner
curl -fsSL https://raw.githubusercontent.com/aifunnels/claudefree/master/scripts/setup-mac.sh | bash
Note
Abra o Terminal, cole a linha acima e pressione Enter.

Funciona em qualquer IDE

IDE

VSCode / Cursor

Abre o terminal integrado (Ctrl+`), cola o one-liner acima. O projeto aparece no Explorer. Agentes ficam em .claude/agents/

IDE

Antigravity

O script ja cria a estrutura .agent/ automaticamente: agents em .agent/skills/, skills em .agent/workflows/, regras em .agent/rules/

CLI

Terminal Puro

PowerShell (Windows) ou Terminal (macOS). Funciona sem IDE — o OpenClaude e um CLI independente.

PS — Providers

Escolha seu provider

Voce nao precisa ficar preso a um unico provider. Centralize tudo em uma plataforma ou misture.

Gratis

Ollama (Local)

100% na sua maquina. Sem API key, sem custo, sem enviar dados. Ideal pra comecar e pra privacidade total.

Recomendado

OpenRouter

1 conta, 1 API key, 200+ modelos: GPT-4o, Claude, Gemini, DeepSeek, Llama, Mistral. Hub unico. Crie a key em openrouter.ai/keys

Alternativa

Apify

Para scraping, extracao de dados e automacao web. Combinado com Ollama ou OpenRouter, cobre 100% das necessidades.

Direto

OpenAI / DeepSeek

Se ja tem API key da OpenAI ou DeepSeek, use direto sem intermediario. Funciona com qualquer endpoint /v1.

Dica
Estrategia recomendada: Use Ollama local pro router (@cos) e tarefas rapidas (custo zero). Use OpenRouter pros agentes criativos (@copy, @cmo, @traffic) que precisam de qualidade alta. Melhor custo-beneficio possivel.

Configuracao rapida — OpenRouter

powershellopenrouter
# Uma key, todos os modelos
$env:CLAUDE_CODE_USE_OPENAI="1"
$env:OPENAI_BASE_URL="https://openrouter.ai/api/v1"
$env:OPENAI_API_KEY="sk-or-SUA-KEY-OPENROUTER"
$env:OPENAI_MODEL="openrouter/free"

openclaude --bare

export OPENAI_BASE_URL=https://openrouter.ai/api/v1
export OPENAI_API_KEY=sk-or-SUA-KEY-OPENROUTER

07 — Known Issues

Corrigir travamento por MCP

Blocker
Se voce ja usa o Claude Code, o OpenClaude le a mesma config em ~/.claude/settings.json e tenta conectar em todos os MCP servers. Servers que precisam de auth ou estao offline travam a inicializacao.
Solucao A
Usar --bare — pula MCPs, hooks, plugins, prefetches
rapido
Solucao B
Settings separado: openclaude --settings ~/.openclaude-settings.json
definitivo
terminalcriar settings limpo
# Windows PowerShell
'{"mcpServers":{}}' | Out-File -Encoding utf8 "$HOME\.openclaude-settings.json"

# macOS / Linux
echo '{"mcpServers":{}}' > ~/.openclaude-settings.json

# Rodar com settings limpo
openclaude --settings ~/.openclaude-settings.json
