01 — Setup

Uma linha. Tudo automatico.

Antes de rodar: crie sua API key gratis em openrouter.ai/keys

O script vai pedir a key no meio do processo, testar, e abrir o OpenClaude automaticamente.

Abra o PowerShell (ou o terminal do VSCode/Cursor) e cole:

powershellwindows
irm https://raw.githubusercontent.com/aifunnels/claudefree/master/scripts/setup-windows.ps1 | iex
NAO use Git Bash. Abre PowerShell ou o terminal do VSCode (Ctrl+`).

Abra o Terminal (ou o terminal do VSCode/Cursor) e cole:

bashmacos / linux
curl -fsSL https://raw.githubusercontent.com/aifunnels/claudefree/master/scripts/setup-mac.sh | bash

O que o script faz

1. Dependencias
Instala Node.js, Git, OpenClaude e ripgrep (pula o que ja tem)
auto
2. API Key
Pede sua key do OpenRouter e testa na hora
1 input
3. Kit PA
Cria pasta minha-empresa na Area de Trabalho com 14 agentes, 14 skills, 50 prompts
auto
4. Abre
Salva a key e abre o OpenClaude automaticamente
auto

Configuracao manual (se nao usou o script)

$env:OPENAI_BASE_URL="https://openrouter.ai/api/v1"
$env:OPENAI_API_KEY="sk-or-SUA-KEY"
$env:OPENAI_MODEL="openrouter/free"
openclaude --bare
